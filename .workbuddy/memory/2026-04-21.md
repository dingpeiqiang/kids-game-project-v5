## 修复：注册时 profile_type NOT NULL 报错

**根因**：`UserServiceImpl.register()` 创建 `UserProfile` 时未设置 `profileType`，数据库字段 NOT NULL 无默认值 → `DataIntegrityViolationException`。

**修复**：根据 `userType` 自动设置 `profileType`：
- `userType == 0` → `KID_INFO`
- 其他 → `PARENT_INFO`

**文件**：`kids-game-backend/kids-game-service/.../UserServiceImpl.java:136`

## 修复：注册提示"token失效或已过期"

**根因**：`SecurityConfig` 的 `permitAll` 白名单缺少注册接口（`/api/user/register` 等），导致注册请求被 Spring Security 拦截，返回 401 "Token 无效或已过期"。

**修复**：在 `SecurityConfig.java` 的 permitAll 白名单中添加：
- `/api/user/register`
- `/api/kid/register`
- `/api/parent/register`

**文件**：`kids-game-backend/kids-game-common/.../SecurityConfig.java`
