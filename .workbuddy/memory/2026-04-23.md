## 排行榜系统实现

**目标**：为 simple-game 游戏实现分数记录和排行榜功能。

### 后端修改

1. **LeaderboardUtil.java** (`kids-game-web/.../util/`)
   - 改为非静态方法，支持 Spring 依赖注入
   - `updateScore()`: 同步更新总榜/日榜/月榜/年榜
   - `getTopList()`: 获取排行榜 TOP N
   - `getUserRank()`: 获取用户排名

2. **LeaderboardService.java** (`kids-game-service/.../service/`)
   - 新增服务层，封装排行榜操作
   - `submitScore()`: 分数上报
   - `getTopList()` / `getDailyTop()` / `getMonthlyTop()` / `getYearlyTop()`: 各类型榜单查询

3. **LeaderboardController.java** (`kids-game-web/.../controller/`)
   - `GET /api/leaderboard/top`: 获取排行榜 TOP N（公开）
   - `GET /api/leaderboard/user-rank`: 获取用户排名（需 token）
   - `POST /api/leaderboard/submit`: 提交分数（需 token）
   - `POST /api/leaderboard/batch-user-rank`: 批量获取排名
   - `GET /api/leaderboard/my-ranks`: 获取我的所有游戏排名

4. **SecurityConfig.java**: 添加 `/api/leaderboard/top` 为公开接口

### 前端修改

1. **leaderboardService.ts** (`simple-game/src/services/`)
   - 新建服务，封装排行榜 API 调用
   - `getTopList()`: 获取排行榜
   - `getUserRank()`: 获取用户排名
   - `submitScore()`: 提交分数
   - `getMyRanks()`: 批量获取我的排名

2. **userService.ts**:
   - `recordGameResult()` 改为 async，同步分数到后端
   - 新增 `getGameLeaderboard()` / `getMyGameRank()` 方法

3. **App.ts**:
   - `endGame()` 改为 async，等待后端同步结果
   - 游戏结束时显示后端返回的真实排名

4. **LeaderboardPanel.vue** (`simple-game/src/components/`)
   - 新建 Vue 组件，展示排行榜面板
   - 支持总榜/日榜/月榜/年榜切换
   - 显示我的排名

### API 端点

| 方法 | 路径 | 说明 | 认证 |
|------|------|------|------|
| GET | /api/leaderboard/top | 获取 TOP N | 否 |
| GET | /api/leaderboard/user-rank | 获取用户排名 | 是 |
| POST | /api/leaderboard/submit | 提交分数 | 是 |
| POST | /api/leaderboard/batch-user-rank | 批量获取排名 | 是 |
