| 问题 | 位置 | 原逻辑 | 修复后 |

## rpgShooterTowerDefense 建造栏费用显示修复（19:40）

**Bug**：建造栏4个炮台按钮的费用/属性全部硬编码（如 `💎 40`），config.ts 改了数值后 UI 不同步。

**修复**：将4个静态按钮改为 `Object.keys(TURRET_CONFIGS).map()` 动态渲染，所有数值从 `TURRET_CONFIGS` 实时读取：
- 费用：`cfg.cost`（不可负担时变红色）
- 属性：`cfg.damage` / `cfg.fireRate` / `cfg.range` / `cfg.special`
- 图标/名称：从 `TURRET_DISPLAY` 读取
- 余额不足时自动半透明 + cursor-not-allowed

## rpgShooterTowerDefense React → Vue 3 转换（19:47）

**文件**：`RpgShooterTD.tsx` → `RpgShooterTD.vue`（新建）
**导出**：`games/index.ts` 改为指向 `.vue` 文件

**转换映射**：
| React | Vue 3 |
|-------|-------|
| `useState` | `ref()` / `reactive()` |
| `useEffect` | `onMounted` + `onBeforeUnmount` |
| `useRef` | `ref()` |
| `className` | `class` |
| JSX 模板 | Vue `<template>` |
| `React.FC` | `<script setup lang="ts">` |

**保留不变**：所有游戏逻辑函数（updateGame/render/drawGrid 等）
