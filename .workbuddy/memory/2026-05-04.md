# 2026-05-04 工作日志

## rpgShooterTowerDefense 游戏优化

### 完成的优化

**1. 易用性改进**
- 简化建造模式切换：B键(桌面)/右上角按钮(通用) 替代 双指点击
- 建造UI优化：网格布局+图标价格，数字键1-4快速选择
- 新增操作引导：开始界面显示操作说明
- 点击炮台出售：简化操作，点击即出售返还50%水晶

**2. 操作流畅度**
- WASD键盘控制：直接控制移动，替代鼠标跟随
- 自动瞄准射击：玩家自动朝向最近敌人射击
- 射击冷却优化：250ms→200ms，响应更跟手

**3. 游戏平衡性**
- 炮台性价比提升：所有炮台成本降低20-33%
- 初始资源增加：水晶150→200，能量50→60
- 波次节奏调整：前期敌人数量减少30%，休息时间增加50%

**4. 手机端优化**
- 虚拟摇杆：左下角40%×40%区域，支持拖拽
- 建造模式快捷按钮：右上角区域一键切换
- 触摸目标增大：炮台按钮更大，方便点击
- 摇杆视觉反馈：动态显示基准圆和操控点
