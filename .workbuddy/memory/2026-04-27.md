# 2026-04-27 工作日志

## simple-game 太空射击游戏适配方案演进

### 第一次：等比缩放 + 黑边
- `Math.min(scaleByHeight, scaleByWidth)` → 宽屏手机有黑边
- 用户反馈：边界溢出

### 第二次：拉伸填充
- Canvas = 全屏尺寸，宽高独立缩放 `ctx.setTransform`
- 用户反馈：画面变形，不好看

### 第三次（最终）：Phaser ScaleManager ENVELOP 模式
- 用 Phaser 3 创建游戏实例，ScaleManager 处理缩放适配
- ENVELOP 模式：等比缩放填满屏幕，裁切超出部分，不变形无黑边
- 通过 CanvasTexture 保留原有 Canvas 2D 绘制代码

**修改文件**：
1. **spaceShooter.ts**：完全重写为 Phaser Scene
   - `SpaceShooterScene extends Phaser.Scene`
   - `preload()`: 创建 `CanvasTexture(400, 600)` 作为渲染目标
   - `create()`: 用 `this.add.image()` 显示 CanvasTexture
   - `update()`: 游戏逻辑 + `renderToCanvas()` 绘制到 CanvasTexture + `texture.refresh()`
   - 输入：Phaser pointer 事件，`screenToLogical()` 坐标转换
   - 安全区：SAFE_L/R/T/B 确保 HUD 不被裁切

2. **App.ts**：spaceShooter 分支简化为 `canvas.innerHTML = ''`

3. **main.css**：添加 `#phaser-space-shooter` 全屏容器样式

**Phaser 配置（最终方案）**：
```js
// 统一用 FIT 模式，等比缩放，画面完整不裁切
// 空白区域用深色背景 (#0a0a1e) 填充，视觉上与游戏背景融为一体
// PC 端：容器限最大 450×800
scale: {
  mode: Phaser.Scale.FIT,
  autoCenter: Phaser.Scale.CENTER_BOTH,
  autoRound: true,
}
```

**架构优势**：
- ✅ 不变形（等比缩放）
- ✅ 画面完整（FIT 不裁切）
- ✅ 空白区域深色背景，视觉一体
- ✅ PC 端画面清晰（限制容器尺寸）
- ✅ 保留原有 Canvas 2D 绘制代码
- ✅ 自动处理 DPI/resize/全屏

**踩坑**：
- 修复 `Phaser is not defined`：index.html 需添加 Phaser CDN `<script>`
- PC 端 ENVELOP → 400×600 放大到 1600px+ 画面模糊 → FIT + 限容器尺寸
- 手机端 ENVELOP → 左右内容被裁切显示不全 → 改为 FIT + 深色背景填充空白

## simple-game 首页游戏显示配置化
