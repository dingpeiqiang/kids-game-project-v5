# 2026-05-02 工作日志

## dragonShooter kill-segment 加速 bug 修复

**问题**：身体节销毁后龙不退反进（看起来像加速）。
**原因**：`updateDragon()` 每帧调用 `rf.update(dt)` 覆盖 `ss.target`，`killDragonSegment()` 的回退在下一帧即被抵消。
**修复**：重构 `updateDragon` v2 — 移除 `rf.update()` 调用，改为自行管理 `ss.target` 推进（`ss.target += pixelSpeed * dt`），RouteFollower 降级为纯位置查询工具。新增 `PIXEL_SPEED_CACHE` 缓存像素速度。
**文件**：`dragonShooter/dragon.ts`
