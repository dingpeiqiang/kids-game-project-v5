## 拼图游戏开发完成（2026-03-29）

**游戏路径**：`kids-game-house/games/puzzle/`

**开发方式**：通过 `init-game.ps1` 初始化，只修改了 `MyGameScene.ts` + `difficulty.json`

**游戏玩法**：
- 图片切割为 N×N 拼块（简单3×3/普通4×4/困难5×5）
- Canvas 绘制渐变拼图图案（无需外部图片）
- 点击与空白相邻的拼块滑动归位
- 全部归位后按用时+步数计算得分

**关键实现**：
- `doSwap()` 交换逻辑：保存原始坐标再交换，避免状态污染
- `shuffle()` 打乱：随机合法滑动 N×40 次，保证有解
- 难度配置读取：`import difficultyConfig from '@/config/difficulty.json'`，按 `gameStore.difficulty` 查找

**tsc --noEmit**：零错误 ✅

## init-game.ps1 编码修复（2026-03-29）

**问题**：脚本用 `Set-Content -Encoding UTF8` 写文件会带 UTF-8 BOM，导致 Node.js 解析 JSON 失败（报 `Unexpected token '锘?`）。

**修复**：改用 `[System.IO.File]::WriteAllText($path, $content, [System.Text.UTF8Encoding]::new($false))` 写入无 BOM 的 UTF-8。

**验证**：`my-puzzle` 游戏（拼图游戏）通过 `init-game.ps1` 完整初始化后，Vite 启动成功，无任何错误。
