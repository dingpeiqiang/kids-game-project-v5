## 暂停状态保持与恢复（上午）

**问题**：暂停期间有 3 个独立时间流仍在继续：
1. `itemTimers` (`game.ts` 的 `setTimeout`) — 道具效果暂停期间继续倒计时
2. `activeEffects[].endTime` — UI 进度条在暂停期间空跑
3. `ItemSystem.spawnTimer` (`setInterval`) — 暂停期间继续生成道具

**修复方案：暂停 = 冻结所有计时器；恢复 = 按剩余时间重建**

**改动文件：**
- `ItemSystem.ts`：新增 `isPaused`, `spawnTimerStartedAt`, `spawnTimerElapsed` 字段；`startSpawnTimer()` 记录启动时间；新增 `pause()` / `resume()` 方法（pause 保存剩余时间并清 setInterval；resume 用 setTimeout 补齐本周期剩余时间后再转 setInterval）
- `stores/game.ts`：新增 `pausedEffectRemaining: Map<string, number>`；重写 `togglePause()`（暂停时从 `activeEffects.endTime` 计算剩余时间并清计时器；恢复时按剩余时间重建 setTimeout 并更新 `endTime`）；`resetItemEffects()` 同时清理 `pausedEffectRemaining`
- `SnakeGame.vue`：`watch isPaused` 中新增 `itemSystem.pause()` / `itemSystem.resume()` 调用

## 暂停按钮无响应 Bug 修复（上午）

**问题**：点击暂停按钮没有反应。

**根本原因**：`PauseButton.vue` 第 17 行使用了解构赋值：
```typescript
const isPaused = gameStore.isPaused  // ❌ 解构丢失了 Pinia ref 的响应式
```
解构 Pinia store 的 ref 属性后，变量变成了普通值，不再是响应式对象，模板无法感知状态变化，按钮显示永远不更新（实际上 togglePause 还是执行了，但视觉没反应可能让用户以为没生效）。

**修复**：改用 `computed` 包装：
```typescript
import { computed } from 'vue'
const isPaused = computed(() => gameStore.isPaused)
```

**文件**：`kids-game-house/games/snake/src/components/ui/PauseButton.vue`

## 暂停被连续触发 N 次 Bug 修复（上午）

**问题**：每次点击暂停，日志里出现 10 次 ⏸️/▶️ 交替，暂停无法正常工作。

**根本原因**：`SnakeGame.vue` 在使用 `<PauseButton>` 组件时，父组件同时绑定了 `@click="gameStore.togglePause()"`：
```html
<PauseButton 
  :isPaused="gameStore.isPaused" 
  @click="gameStore.togglePause()"   ← 父组件监听了一次
/>
```
而 `PauseButton` 内部按钮自己的 `@click="toggle"` 也调用了一次 `gameStore.togglePause()`，导致每次点击触发两次。多次进出页面后事件监听器叠加，最终连续触发 N 次。

**修复**：删除父组件的 `:isPaused` prop 和 `@click` 监听，`PauseButton` 完全自治：
```html
<PauseButton />
```

**文件**：`kids-game-house/games/snake/src/components/game/SnakeGame.vue`

## 恢复游戏后立即 GameOver Bug 修复（上午）

**根本原因**：`startGameLoop` 的 rAF loop 里，暂停时 `lastTime` 不更新。恢复后第一帧 `deltaTime = (当前时间 - 暂停前时间) / 1000`，等于整个暂停时长（可能几秒），`moveSnake` 一次步进极大距离，蛇直接撞墙 → GameOver。

**修复**：在 `loop()` 的 `else if (isPaused)` 分支里持续刷新 `lastTime = currentTime`，保证恢复后第一帧 deltaTime 正常。

```typescript
} else if (gameStore.isPaused) {
  // 暂停时持续刷新 lastTime，防止恢复后 deltaTime 暴增
  lastTime = currentTime
}
```

**文件**：`kids-game-house/games/snake/src/components/game/SnakeGame.vue`
