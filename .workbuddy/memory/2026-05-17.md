## simple-game 收藏页游戏卡片动画修复

**问题**：`我的收藏` 页面切换后，游戏卡片的 Canvas 预览动画无法正常播放。

**根因**：
- `favoritesContent` 初始 `display:none`，切换 `display:block` 后立即执行 `setTimeout(50ms)` 触发 `renderPreview`
- `renderPreview` 检测 Canvas 的 `getBoundingClientRect()` 尺寸仍为 0（浏览器布局未完成），进入重试但体验差
- 卡片本身也没有入场动画（CSS 无 fadeIn/slideIn）

**修复内容（App.ts + main.css）**：
1. 将 `setTimeout(50ms)` 替换为 `requestAnimationFrame(() => requestAnimationFrame(...))` 双帧等待，确保布局完成
2. 每张卡片的预览启动增加 `i * 30ms` 错落延迟
3. CSS 新增 `@keyframes cardFadeIn`（淡入+上移+scale），`.game-card` 默认带此动画
4. 收藏页/搜索页每张卡片设 `animationDelay = index * 55ms`，实现错落入场效果
5. `.search-header/.favorites-header` 也加 `cardFadeIn` 动画
6. 搜索结果页同步修复相同问题
