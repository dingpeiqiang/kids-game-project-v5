## 太空射击道具叠加系统改造

**修改文件：**
- `kids-game-house/games/simple-game/src/games/spaceShooter.ts`

**改动内容：**
- 移除 `MAX_ACTIVE_BUFFS=3` 限制和 `activeBuffOrder` FIFO 队列
- 每个 buff 类型维护独立计时器栈（`tripleShotTimers[]` 等），每个拾取增加一个栈
- 特效强度随栈数叠加：
  - 三连：弹道数 = 3 × stack数
  - 散射：每侧步数 = 3 + (stack-1) × 2，角度收紧
  - 速射：CD = SHOOT_CD / (1 + stack × 0.5)
  - 击穿：穿透次数 = stack × 2
  - 激光：爆发次数 = 3 + stack × 2
- `stepBuffTimers()` 方法独立递减每个栈，到期自动移除一个栈；全部到期才重置
- HUD 显示 stack 数（如 `⚡三连 x3 5s`）
