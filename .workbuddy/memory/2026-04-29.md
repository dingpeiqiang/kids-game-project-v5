# 2026-04-29 日记

## dragonShooter 游戏优化

### 战斗平衡性重做
- 子弹伤害：2 → 1（需要更多命中）
- 射速：120ms → 80ms（更快）
- 小龙 HP：1 → 2
- 龙速整体提升约 60%
- levelTarget：6 → 4（前期快速升级）
- maxDragons：3 → 5

### 核心机制改动
- **启用龙体分裂**：击中龙身后从该节断裂成两段，后半段成为独立小龙
- **新增龙体碰撞**：任何龙节碰到玩家都造成伤害（1血），受伤后1.5秒无敌
- **BOSS触底秒杀**：长龙（>15节）或BOSS触底直接失败

### 蜿蜒路径优化
- 单频率波动 → 双频率叠加（wave1 + wave2）
- 更自然、更不可预测的游动轨迹

### 龙生成优化
- 生成概率提升：0.03 → 0.06
- 特殊龙门槛降低（coin从4→2，elite从10→6）
