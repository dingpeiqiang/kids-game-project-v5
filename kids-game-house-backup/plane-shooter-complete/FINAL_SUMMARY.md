### 1. 程序化图像生成技术

**Sharp 技术应用**:
```javascript
// 高性能像素操作 + Sharp 转换
const buffer = Buffer.alloc(width * height * 4);
// 填充 RGBA 数据
for (let y = 0; y < height; y++) {
  for (let x = 0; x < width; x++) {
    const color = drawFunc(x, y, w, h);
    const idx = (y * width + x) * 4;
    buffer[idx] = color.r;
    buffer[idx + 1] = color.g;
    buffer[idx + 2] = color.b;
    buffer[idx + 3] = color.a;
  }
}
await sharp(buffer, { raw: { width, height, channels: 4 } })
  .png()
  .toFile(filepath);
```

### 2. WAV 音频算法合成

**核心技术**:
```javascript
// 波形生成
sample = Math.sin(2 * Math.PI * frequency * t);  // 正弦波
sample = Math.sign(Math.sin(2 * Math.PI * frequency * t));  // 方波
sample = Math.random() * 2 - 1;  // 噪声

// 特殊音效
const noise = (Math.random() * 2 - 1) * Math.exp(-t * 5);
const lowFreq = Math.sin(2 * Math.PI * 80 * t) * Math.exp(-t * 3);
sample = (noise + lowFreq) * 0.8;  // 爆炸音效
```
