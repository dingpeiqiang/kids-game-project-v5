## 一、游戏概述

### 1.1 游戏简介
飞机大战是一款经典的纵向卷轴射击游戏。玩家控制一架战斗机，在太空中与敌机作战，躲避敌方子弹并消灭所有敌人。

### 1.2 目标用户
- 年龄段：7-15 岁儿童
- 游戏经验：初级到中级
- 设备：手机、平板、PC

### 1.3 核心玩法
- **移动控制**: 玩家通过触摸/鼠标/键盘控制飞机移动
- **自动射击**: 飞机自动发射子弹
- **躲避攻击**: 避开敌机和敌方子弹
- **收集道具**: 获取强化道具提升战力

## 二、游戏设计细节

### 2.1 游戏世界参数

```javascript
// 游戏画布尺寸（竖屏）
GAME_WIDTH = 720
GAME_HEIGHT = 1280

// 网格系统（用于对齐和碰撞检测）

```

### 2.2 游戏对象设计

#### 2.2.1 玩家飞机
```javascript
player: {
  width: 60,        // 飞机宽度
  height: 80,       // 飞机高度
  speed: 10,        // 移动速度
  health: 3,        // 生命值
  bulletType: 1,    // 子弹类型：1=单发，2=双发，3=散射
  fireRate: 200     // 射击间隔 (ms)
}
```

#### 2.2.2 敌方飞机

| 类型 | 宽度 | 高度 | 速度 | 生命 | 得分 | 行为模式 |
|------|------|------|------|------|------|----------|
| **小型敌机** | 40 | 40 | 5 | 1 | 100 | 直线向下 |
| **中型敌机** | 50 | 60 | 3 | 3 | 300 | 左右摆动 |
| **大型敌机** | 80 | 80 | 2 | 10 | 1000 | 追踪玩家 |
| **Boss** | 150 | 150 | 1 | 50 | 5000 | 复杂弹幕 |

#### 2.2.3 子弹类型

**玩家子弹**:
- 等级 1: 单发直线子弹 (伤害 1)
- 等级 2: 双发平行子弹 (伤害 1)
- 等级 3: 散射三向子弹 (伤害 1)

**敌方子弹**:
- 普通子弹：直线向下，速度 8
- 追踪子弹：追踪玩家，速度 6
- 散射子弹：多方向，速度 5

## 四、游戏规则

### 4.1 得分系统
- 击落小型敌机：+100 分
- 击落中型敌机：+300 分
- 击落大型敌机：+1000 分
- 击落 Boss: +5000 分
- 连续击杀奖励：每 10 连击额外 +500 分

### 4.2 难度级别
- **简单**: 敌机速度慢，子弹少，玩家生命 5
- **普通**: 敌机速度正常，子弹中等，玩家生命 3
- **困难**: 敌机速度快，子弹密集，玩家生命 2

### 4.3 游戏结束条件
- 玩家生命值归零
- 玩家被敌机撞击

### 4.4 胜利条件
- 在规定时间内存活
- 击落指定数量的敌机
- 击败 Boss

## 五、技术实现要点

### 5.1 物理引擎
- 使用 Arcade Physics 进行碰撞检测
- AABB 碰撞盒：略小于 sprite 视觉尺寸（约 80%）
- 子弹使用圆形碰撞检测

### 5.2 性能优化
- 对象池管理子弹和敌机
- 视锥体剔除：只渲染屏幕内的对象
- 粒子效果限制最大数量

### 5.3 适配策略
- 响应式画布：自动适配不同屏幕尺寸
- 触控优先：移动端优化操作体验
- 60FPS 目标：保持流畅游戏体验

## 六、UI 界面设计

### 6.1 游戏内 UI
- **顶部栏**: 分数、生命值、波次
- **暂停按钮**: 右上角
- **血量条**: 玩家和 Boss 上方显示

### 6.2 游戏外 UI
- **开始界面**: 游戏标题、开始按钮、操作说明
- **暂停界面**: 继续、重新开始、退出选项
- **结束界面**: 最终得分、星级评价、再来一局按钮

## 七、扩展性设计

### 7.1 关卡系统
- 每关有不同的敌机配置
- 关卡难度递增
- 每 5 关出现 Boss

### 7.2 成就系统
- 击落 100 架敌机
- 连续命中 50 次
- 无伤通关
- 最高分记录

### 7.3 主题系统
- 支持自定义飞机皮肤
- 支持自定义子弹样式
- 支持自定义背景主题
