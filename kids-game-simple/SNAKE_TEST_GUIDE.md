```bash
cd kids-game-house/games/simple-game
npm run dev
```

打开浏览器访问: http://localhost:5103

在游戏首页找到并点击"贪吃蛇"游戏卡片

## 🐛 常见问题排查

### 问题1: 游戏立即结束
**症状**: 进入游戏后蛇立即死亡
**检查**: 
- 确认 `alive` 初始值为 `false`
- 确认 `gameStarted` 初始值为 `false`
- 确认点击/按键事件能正确触发游戏开始

### 问题2: 蛇不移动
**症状**: 点击开始后蛇不动
**检查**:
- 确认 `gameStarted` 已设置为 `true`
- 确认 `alive` 已设置为 `true`
- 检查游戏循环中的移动条件

### 问题3: 控制无效
**症状**: 点击或按键后蛇不转向
**检查**:
- 确认事件监听器已正确添加
- 确认方向判断逻辑正确
- 检查是否有阻止默认行为
