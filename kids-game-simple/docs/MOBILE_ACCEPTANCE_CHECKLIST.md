## 五、已知代码层待验 / 风险（优先测）

| id | 风险说明 |
|----|----------|
| wangzheRpg | `endGame()` 可能未调用外壳 `onEnd()`，需验「结算后能否回大厅」 |
| rpgShooter | 游戏结束点击会 `location.reload()`，在 WebView 中行为可能不符合预期 |
| dragonShooter | 移动端 body 级 wrapper，需验与大厅切换、销毁 |
| contraRpg | `touchToGame` 与横屏旋转，边缘设备易偏 |
| towerDefense | 确认 `teardownInput` 与 import 完整、能编译运行 |
| plantsVsZombies | 确认 `bindCanvasPointerInput` 等 import 完整 |
