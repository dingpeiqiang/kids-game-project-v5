# 市场小游戏参考库（策划对标用）

> **用途**：本仓库**尚无**可作为「填表金样例」的定稿游戏；新游戏策划应对标**外部市场**里体验成熟的产品，在 GDD **§1 文档信息**与 **§1.2 竞品拆解**中写明「抄什么 / 不抄什么」。  
> **注意**：参考的是 **节奏、反馈、数值密度、留存结构**，不是抄袭 IP、美术或完整关卡；儿童向须满足 [GDD 模板 §4.8](GDD-AI-ASSET-TEMPLATE.md) 合规。

---

## 1. 怎么用（与模板绑定）

| 步骤 | 做法 |
|------|------|
| 选题 | 从下表选 **1 主类型 + 1～2 款参考产品**（可写微信/抖音小游戏名或玩法统称） |
| 拆解 | 在 `docs/<gameId>-GDD.md` 填：**抄机制**（可量化）、**不抄**（包体/社交/变现/题材） |
| 反推 | 用参考品的「首局 30s」反推本游戏 **§3 时间轴**、**§5 事件**（比参考更简或更儿童） |
| 验收 | **§2 目标值**要对标参考品的爽点频率（如：首次正向反馈 ≤8s），数值在 **§4** 落地 |

**禁止**：只写「参考某某很火」而不写机制拆解 → 视为未填 §1。

## 2. 按类型选参考（体验维度，非 IP 清单）

### 2.1 竖屏射击 / 割草 / 断节龙类（解压 + 动作）

| 市场共性（值得抄） | 建议不抄 |
|--------------------|----------|
| 自动开火 + 单指走位，3s 内必有一次命中 | 满屏弹幕、恐怖生物 |
| 击杀 → 短时停 + 掉落 → 三选一成长，约 20～40s 一轮 | 复杂天赋树、局外付费 |
| 段血/波次可视化，断节有「中」级视听反馈 | 写实血腥、枪械军事风 |

**可对标的玩法统称**：断节龙/弹幕幸存者类、竖屏 Roguelite 射击（儿童向需弱化难度与视觉刺激）。

**GDD 重点**：§4 射速/伤害/段血、§5 `hit_light`/`kill_seg`/`pick_buff`、§3 首次断节时刻。

---

### 2.2 塔防 / 防线类（策略 + 解压）

| 市场共性 | 建议不抄 |
|----------|----------|
| 第一波前 10～20s 内完成第一次建造并看到击杀 | 多线出兵、复杂经济 |
| 波次间隙给「喘息 + 升级」；失败原因明确（漏怪数） | 长局 15min+、PVP |
| 塔种类 ≤5，每种 1 个核心差异（减速/范围/单点） | 写实战争、丧尸 |

**玩法统称**：单路径塔防、欢乐防线类、植物防线类（儿童向用萌物替代对抗叙事）。

**GDD 重点**：§4 波表 + 造价、§3 首次造塔与首次漏怪、§5 `tower_place`/`wave_clear`。

---

### 2.3 消除 / 棋盘类（益智 + 解压）

| 市场共性 | 建议不抄 |
|----------|----------|
| 5s 内第一次消除；连击有递进音效与飘字 | 赌博式复活、分享胁迫 |
| 关卡目标单一可见（步数/收集/分数） | 无限卡关逼氪 |
| 「差一点就过」的 1～2 星阈值 | 成人向题材 |

**玩法统称**：三消/点消/合成棋盘（儿童向控制单局时长 3～5min）。

**GDD 重点**：§4 棋盘尺寸与计分、§5 `match`/`combo`/`level_win`。

---

### 2.4 跑酷 / 躲避 / 球类（反应 + 竞技）

| 市场共性 | 建议不抄 |
|----------|----------|
| 3s 内完成第一次左右/跳跃操作 | 速度曲线过早劝退 |
| 距离/分数即成绩；撞障死亡原因一眼懂 | 恶意碰撞盒 |
| 单局 1～3min；复活可选且儿童向无惩罚焦虑 | 恐怖障碍、广告复活链 |

**玩法统称**：无尽跑酷、滚球/跳台、3D 轻物理（WebView 注意性能）。

**GDD 重点**：§4 速度曲线与间距、§3 首次操作与首次撞障教学。

---

### 2.5 轻对战 / 1v1 / IO（竞技）

| 市场共性 | 建议不抄 |
|----------|----------|
| 1min 内必有第一次有效交互（命中/占点） | 5v5、语音、段位辱骂 |
| 胜负由操作可见；AI 难度阶梯写在 §4 | 随机秒杀、氪金数值 |
| 自动普攻 + 少量手动技能 | 复杂出装、长文案技能 |

**玩法统称**：迷你 MOBA、竞技场 1v1、轻度乱斗（儿童向：软失败、无恐怖击杀）。

**GDD 重点**：§4 技能 CD/AI 血量、§5 命中与击杀分层反馈。

## 3. 跨类型「高质量」共性（P0 对标尺）

以下用于填 **§2 验收指标**，不依赖某一款具体产品：

| 维度 | 市场成熟小游戏常见下限 | 建议写入 §2 的指标示例 |
|------|------------------------|-------------------------|
| 首局上手 | 无长文，≤60s 进入核心循环 | 首次核心操作 ≤5s |
| 首次爽点 | 首次「赢一点」≤10～15s | 首次击杀/消除/造塔 ≤\_\_s |
| 反馈密度 | 轻反馈持续；中反馈每 20～40s | 每 \_\_s 至少 1 次「中」级 event |
| 单局时长 | 休闲合集常见 2～5min | 中位 \_\_～\_\_min |
| 失败 | 原因可枚举；可立即重开 | 死因 ≤3 种文案 |
| 成绩 | 本地纪录或星级 | 3★ 阈值写进 §4.9 |
| 视听 | BGM 循环不刺耳；重特效不常开 | 同时「重」特效 ≤1 |

## 4. 文档信息栏（每款游戏必填）

在 `docs/<gameId>-GDD.md` 顶部增加：

| 字段 | 示例 |
|------|------|
| **主参考产品** | 「竖屏 Roguelite 射击类（儿童向弱化）」或具体小游戏名 |
| **抄机制** | 自动射击 + 断节掉落三选一 |
| **不抄** | 局外养成深度、恐怖美术、社交排行 |
| **参考首局节奏** | 参考品约 8s 首次击杀 → 本作目标 ≤8s |
