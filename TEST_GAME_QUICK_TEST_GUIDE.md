## 💡 常见问题排查

### 问题 1：内容不居中
**检查**：
1. App.vue 是否有 `display: flex; flex-direction: column;`
2. 视图组件是否有正确的 class
3. CSS 是否有冲突

**解决**：确保父容器建立了 Flex 上下文

### 问题 2：字体大小不对
**检查**：
1. `useResponsiveUI` 是否正确初始化
2. `uiScale` 值是否正确
3. 计算结果是否符合预期

**解决**：在控制台打印 `ui.uiScale` 值

### 问题 3：动画不流畅
**检查**：
1. 浏览器性能
2. CSS 动画定义是否正确
3. 是否有其他样式覆盖

**解决**：检查 DevTools 的动画面板
