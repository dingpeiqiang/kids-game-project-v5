## 🎯 功能操作流程

### **用户详情查看**
```
1. 点击列表中的"详情"按钮
   ↓
2. 打开用户详情对话框
   ↓
3. 显示用户完整信息
   ↓
4. 可执行操作：
   - 关闭对话框
   - 编辑用户信息
   - 重置密码
```

---

### **编辑用户信息**
```
1. 在详情对话框点击"编辑信息"
   ↓
2. 打开编辑对话框，表单预填充当前数据
   ↓
3. 修改字段：
   - 昵称（必填）
   - 头像 URL（可选）
   - 疲劳点（0-100）
   - 每日答题积分（0-100）
   ↓
4. 点击"保存"提交
   ↓
5. 成功后刷新列表
```

---

### **重置密码**
```
1. 在详情对话框点击"重置密码"
   ↓
2. 打开重置密码对话框
   ↓
3. 显示安全提示：
   ⚠️ 重置后用户密码将恢复为默认密码：123456
   ↓
4. 确认重置
   ↓
5. 成功后显示提示消息
```
