### **场景 A：快速选择头像**
```
1. 点击"选择头像"按钮
   ↓
2. （未来）打开头像选择器
   ↓
3. 浏览并选择喜欢的头像
   ↓
4. 点击确认
   ↓
5. 头像自动填充到 URL 框
   ↓
6. 左侧实时预览
```

### **场景 B：使用推荐头像**
```
1. 查看"常用头像"区域
   ↓
2. 点击某个圆形头像
   ↓

   ↓
4. 该头像显示蓝色边框
```

### **场景 C：上传本地头像**
```
1. 点击"上传头像"按钮
   ↓
2. （未来）打开文件选择对话框
   ↓
3. 选择本地图片
   ↓

   ↓

### **场景 D：恢复默认**
```
