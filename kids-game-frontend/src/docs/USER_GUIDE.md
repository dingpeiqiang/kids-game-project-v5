# 项目手册使用指南

## 📖 欢迎使用项目手册功能！

项目手册是管理员中心的文档查看工具，帮助您快速浏览和检索项目的所有技术文档。

---

## 🚀 快速上手

### 第一步：进入项目手册

1. 访问管理后台：http://localhost:5173/admin/dashboard
2. 点击左侧菜单的 **"📚 项目手册"**
3. 进入文档查看器界面

### 第二步：浏览文档

**方式一：按分类浏览**
- 左侧边栏显示所有文档分类
- 点击分类下的文档标题即可查看详情
- 当前阅读的档会高亮显示

**方式二：搜索文档**
- 在顶部搜索框输入关键词
- 系统自动过滤显示匹配的文档
- 点击搜索结果查看内容

### 第三步：阅读文档

- 右侧主内容区显示 Markdown 渲染结果
- 支持代码高亮、表格、列表等格式
- 可以复制文档内容

## ⚙️ 自定义配置

### 添加新文档

**方法一：手动添加**

1. 在 `src/docs/` 目录创建新文件：
   ```markdown
   # 我的文档
   
   这是文档内容...
   ```

2. 编辑 `DocViewer.vue`，在 `categories` 中添加配置：
   ```typescript
   {
     id: 'my-doc',
     title: '我的文档',
     icon: '📄',
     path: '/src/docs/my-doc.md'
   }
   ```

**方法二：批量导入**（阶段三实现）
- 自动扫描 docs 目录
- 根据文件名生成配置
- 支持自定义元数据

### 修改分类结构

编辑 `DocViewer.vue` 中的 `categories` 数组：

```typescript
const categories = ref([
  {
    name: '新分类',
    icon: '🆕',
    docs: [/* 文档列表 */]
  }
])
```

### 调整文档顺序

修改 `docs` 数组中文档的顺序即可改变显示顺序。

## 🎯 最佳实践

### 1. 文档组织原则

✅ **推荐**
- 一个功能一个文档
- 清晰的层级结构
- 统一的命名规范
- 定期更新维护

❌ **避免**
- 文档过大（超过 1000 行）
- 内容混杂无分类
- 命名随意不规范
- 长期不更新

### 2. 搜索技巧

**高效搜索**
- 使用核心关键词
- 尝试同义词搜索
- 结合分类筛选

**搜索示例**

|------|--------|------|
| 找启动方法 | 启动、start | 快速启动指南 |

| 找 Bug 修复 | fix、修复 | 各类修复记录 |

## 🐛 常见问题

### Q1: 文档加载失败怎么办？

**A**: 按以下步骤排查：
1. 检查文档路径是否正确
2. 确认文件存在于 `src/docs/` 目录
3. 查看浏览器控制台错误信息
4. 刷新页面重试

### Q2: Markdown 格式显示异常？

**A**: 可能的原因：
- Markdown 语法错误
- 特殊字符未转义
- 代码块缺少语言标识

**解决方法**:
1. 使用 Markdown 编辑器检查
2. 参考其他文档的格式
3. 使用在线 Markdown 验证工具

### Q3: 如何打印或导出文档？

**A**: 
- **打印**: 使用浏览器打印功能（Ctrl+P）
- **复制**: 选中文本复制到剪贴板
- **下载**: 直接访问 `.md` 文件下载（阶段三支持）

### Q4: 搜索不到想要的文档？

**A**: 
1. 尝试不同的关键词
2. 检查文档是否在列表中
3. 确认搜索词拼写正确
4. 可能需要全文搜索功能（阶段二）
