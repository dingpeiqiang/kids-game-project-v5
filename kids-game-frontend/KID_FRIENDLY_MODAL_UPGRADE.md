## 🚀 使用示例

### 基础用法

```vue
<KidFriendlyModal
  v-model="show"
  title="温馨提示"
  content="这是一个可爱的弹窗"
  type="success"
  @confirm="handleConfirm"
/>
```

### 高级用法

```vue
<KidFriendlyModal
  v-model="show"
  title="奖励领取"
  type="celebrate"
  size="lg"
  icon="🎮"
  confirm-text="太棒了！"
  :show-cancel="false"
>
  <div class="reward-content">
    <h4>🎁 恭喜完成挑战！</h4>
    <ul>
      <li>⭐ 金币 x 100</li>

  </div>
</KidFriendlyModal>
```
