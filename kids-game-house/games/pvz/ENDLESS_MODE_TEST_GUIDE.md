## 🐛 常见问题排查

### 问题1：僵尸不生成
**检查**：
- 控制台是否有错误信息
- 确认游戏已开始（倒计时结束）
- 检查 `this.gameStarted` 是否为 `true`

### 问题2：难度不提升
**检查**：
- 等待至少30秒
- 查看控制台是否有 `checkDifficultyIncrease` 相关日志
- 确认 `this.survivalTime` 在递增

### 问题3：阳光数值不对
**检查**：
- 天空阳光应该是 +50
- 向日葵阳光应该是 +75
- 查看 `Sun` 模型的 `value` 属性

### 问题4：UI显示异常
**检查**：
- 右侧是否显示 ⏱️ 💀 📊 三个图标
- 数值是否正常更新
- 浏览器控制台是否有错误

## ✨ 优化建议记录

如果在测试中发现以下问题，可以进一步调整：

1. **难度曲线太平缓** → 减少难度提升间隔（从30秒改为20秒）
2. **后期太难** → 降低生成间隔递减幅度（从300ms改为200ms）
3. **阳光不够用** → 进一步缩短向日葵产出间隔（从12秒改为10秒）
4. **植物太弱** → 增加植物生命值或攻击力
