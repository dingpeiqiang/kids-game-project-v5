# 植物大战僵尸 - 无尽模式版

## 🎮 游戏简介
基于 Phaser 3 开发的植物大战僵尸游戏，现已升级为**无尽生存模式**！

### ✨ 新特性
- ♾️ **无尽模式**：僵尸持续生成，没有终点
- 📈 **动态难度**：每30秒自动提升难度等级
- ⏱️ **生存挑战**：比拼谁能活得更久
- 💀 **击杀统计**：实时显示击杀数和生存时间
- ⚖️ **优化平衡**：更快的节奏，更流畅的体验

---

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 访问游戏
在浏览器中打开：`http://localhost:3004/`

## 🎯 游戏玩法

### 基础操作
1. **收集阳光**：点击掉落的阳光（天空掉落 +50，向日葵产出 +75）
2. **选择植物**：点击顶部卡片栏的植物
3. **种植植物**：点击草坪格子放置植物
4. **铲除植物**：点击铲子按钮，再点击要移除的植物
5. **暂停游戏**：按 ESC 键

### 无尽模式策略
- **早期（Lv.1-3）**：快速建立向日葵经济，布置豌豆射手防御
- **中期（Lv.4-6）**：混合使用寒冰射手控制节奏，坚果墙阻挡
- **后期（Lv.7+）**：大量部署双发射手，合理使用樱桃炸弹清场

### 植物介绍
| 植物 | 阳光消耗 | 冷却时间 | 特点 |
|------|---------|---------|------|
| 🌻 向日葵 | 50 | 5秒 | 产生阳光（+75） |
| 🌱 豌豆射手 | 100 | 5秒 | 基础攻击植物 |
| ❄️ 寒冰射手 | 175 | 5秒 | 减速僵尸 |
| 🔫 双发射手 | 200 | 5秒 | 双倍火力 |
| 🍒 樱桃炸弹 | 150 | 20秒 | 范围爆炸 |
| 🥔 土豆雷 | 25 | 20秒 | 一次性陷阱 |
| 🥜 坚果墙 | 50 | 20秒 | 高血量防御 |

## 📊 难度系统

### 难度等级 progression
```
Lv.1 (0-30s):   普通僵尸
Lv.2 (30-60s):  + 路障僵尸
Lv.3 (60-90s):  更多路障僵尸
Lv.4 (90-120s): + 铁桶僵尸
Lv.5 (120-150s):+ 报纸僵尸
Lv.7+ (180s+):  高级僵尸比例增加
...
持续挑战直到防线被突破！
```

### 典型成绩参考
- **新手**：生存1-2分钟，击杀10-20个僵尸，达到Lv.3-4
- **熟练**：生存3-5分钟，击杀40-60个僵尸，达到Lv.6-8
- **高手**：生存8分钟以上，击杀100+僵尸，达到Lv.15+
