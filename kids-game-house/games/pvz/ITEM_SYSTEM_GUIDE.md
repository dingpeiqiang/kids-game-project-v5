# 植物大战僵尸 - 解压道具系统

## 🎁 道具系统概述

为了让无尽模式更加解压和有趣，我们添加了**随机道具掉落系统**。每45秒会在游戏区域随机生成一个强力道具，帮助玩家应对越来越强的僵尸潮。

---

## ✨ 道具类型

### 1. ❄️ 全屏冻结 (Freeze Item)
**概率**: 25%  
**效果**: 冻结屏幕上所有僵尸10秒

- 所有僵尸停止移动
- 僵尸变为冰蓝色半透明状态
- 10秒后自动解冻恢复

**使用场景**: 
- 僵尸数量过多时紧急控场
- 争取时间重新布置防线
- 配合樱桃炸弹清场

---

### 2. 💰 双倍阳光 (Double Sun Item)
**概率**: 30%  
**效果**: 接下来30秒内所有阳光产出翻倍

- 天空阳光: 50 → 100
- 向日葵阳光: 75 → 150
- 持续30秒

**使用场景**:
- 快速积累阳光
- 紧急补充经济
- 为大规模部署做准备

---

### 3. 🌿 植物强化 (Plant Boost Item)
**概率**: 25%  
**效果**: 所有现有植物血量翻倍，持续20秒

- 立即强化场上所有植物
- 植物显示绿色光晕
- 20秒后恢复原血量

**使用场景**:
- 防线压力大时增强防御
- 保护关键植物（如向日葵）
- 应对高强度僵尸波次

---

### 4. 🍒 樱桃炸弹雨 (Cherry Bomb Item)
**概率**: 20%  
**效果**: 在随机3个空位放置樱桃炸弹

- 自动寻找空闲格子
- 分3次放置，每次间隔0.3秒
- 樱桃炸弹会立即爆炸消灭周围僵尸

**使用场景**:
- 清理密集僵尸群
- 紧急救场
- 快速消灭高级僵尸

---

## 🎮 道具生成机制

### 生成规则
- **首次生成**: 游戏开始后15秒
- **后续生成**: 每45秒生成一个
- **生成位置**: 游戏区域内随机位置（避开边缘）
- **存在时间**: 8秒后自动消失

## 🎯 收集与使用

### 收集方式
1. 道具出现时会浮动并带有彩色光晕
2. 点击道具即可收集
3. 收集后立即生效，无需手动激活
4. 8秒内未收集则自动消失

### UI显示
- **左下角**: `🎁 X` - 显示已收集道具数量
- **游戏结束**: 统计总收集道具数

### 视觉反馈
- **收集动画**: 道具放大并淡出
- **效果提示**: 屏幕中央显示效果说明
- **光晕效果**: 每种道具颜色不同
  - 冻结: 青色光晕 ❄️
  - 双倍阳光: 金色光晕 💰
  - 植物强化: 绿色光晕 🌿
  - 樱桃炸弹: 红色光晕 🍒

## 🔧 技术实现

### 文件结构
```
src/models/
├── Item.js              # 道具基类
├── FreezeItem.js        # 全屏冻结道具
├── DoubleSunItem.js     # 双倍阳光道具
├── PlantBoostItem.js    # 植物强化道具
└── CherryBombItem.js    # 樱桃炸弹道具
```

### 核心代码

#### 道具生成
```javascript
spawnRandomItem() {
  // 随机位置
  const x = Phaser.Math.Between(GL + 50, GL + GRID_W - 50)
  const y = Phaser.Math.Between(GY + 50, GY + GAME_H - 50)
  
  // 加权随机选择类型
  const itemTypes = [
    { type: 'freeze', weight: 25 },
    { type: 'double_sun', weight: 30 },
    { type: 'plant_boost', weight: 25 },
    { type: 'cherry_bomb', weight: 20 }
  ]
  
  const selectedItem = this.weightedRandom(itemTypes)
  
  // 创建对应道具
  switch (selectedItem.type) {
    case 'freeze': new FreezeItem(this, x, y); break
    case 'double_sun': new DoubleSunItem(this, x, y); break
    // ...
  }
}
```

#### 双倍阳光逻辑
```javascript
// Sun.js - collect() 方法
let sunValue = this.value || this.sunValue || 25

// 双倍阳光模式
if (this.scene.doubleSunMode) {
  sunValue *= 2
}

this.scene.addSun(sunValue)
```

## 📊 平衡性设计

### 生成频率
- **45秒间隔**: 既不会太频繁导致依赖，也不会太稀少失去意义
- **15秒首生**: 让玩家早期就能体验到道具系统

### 效果强度
- **冻结10秒**: 足够应对紧急情况，但不会让游戏太简单
- **双倍30秒**: 显著的经济提升，但有时间限制
- **强化20秒**: 短期防御爆发，需要合理使用
- **3个炸弹**: 强力清场，但位置随机有一定风险

### 概率分布
- **双倍阳光30%**: 最常用，帮助经济发展
- **冻结/强化各25%**: 平衡的防御选项
- **樱桃炸弹20%**: 稀有但强力的进攻选项
