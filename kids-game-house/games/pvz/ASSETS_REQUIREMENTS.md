## 📐 技术规范

### 图片格式要求
- **格式**: PNG（支持透明通道）
- **色彩模式**: RGBA8888
- **压缩**: 无损压缩
- **精灵图集**: 使用 TexturePacker 或类似工具打包

### 推荐图集组织
```
sprites_plants.png/json      - 所有植物
sprites_zombies.png/json     - 所有僵尸
sprites_projectiles.png/json - 投射物
sprites_ui.png/json          - UI 元素
sprites_effects.png/json     - 特效
sprites_backgrounds.png/json - 背景元素
```

### 动画帧率
- **标准动画**: 12-15 FPS
- **快速动画**: 20-24 FPS（爆炸、击中）
- **慢速动画**: 8-10 FPS（摇摆、漂浮）

### 命名规范
```
{type}_{action}_{frame_number}.png
示例:
- peashooter_idle_01.png
- zombie_walk_05.png
- sun_fall_03.png
- explosion_08.png
```

## 💡 素材获取建议

### 方案 A：使用现有资源（快速启动）
1. 放大现有 sprites.png 中的素材（2x 或 3x）
2. 使用 AI 图像生成工具补充缺失帧
3. 从开源游戏资源网站获取免费素材

### 方案 B：定制创作（最佳质量）
1. 聘请像素艺术家创作原创素材
2. 使用 Aseprite、Photoshop 等专业工具
3. 保持统一的像素艺术风格

### 方案 C：混合方案（平衡）
1. 核心植物/僵尸使用定制素材
2. 次要元素使用免费资源
3. 特效使用粒子系统动态生成

### 推荐资源网站
- OpenGameArt.org - 免费游戏素材
- Itch.io Game Assets - 独立游戏资源
- Kenney.nl - 高质量免费素材包
- CraftPix.net - 付费优质素材
