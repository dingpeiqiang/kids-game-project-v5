# 🎛️ 游戏配置功能 - 快速使用指南

**5 分钟上手** - 自定义你的专属贪吃蛇体验！

---

## 🚀 快速开始

### Step 1: 打开配置界面

在 StartView 主界面，找到音效和主题选择器下方：

```
🎮 开始游戏
⚙️ 音效开关
🎨 主题选择
⚙️ 游戏配置  ← 点击这里！
```

### Step 2: 调整难度

在弹窗的"🎯 难度设置"区域选择适合你的难度：

| 难度 | 速度 | 适合人群 |
|------|------|----------|
| 😊 **简单** | 150 px/s | 新手玩家 |
| 🙂 **普通** | 200 px/s | 一般玩家（推荐） |
| 😈 **困难** | 300 px/s | 高手玩家 |
| 👿 **极限** | 400 px/s | 挑战自我 |

**操作**: 点击对应难度按钮即可

### Step 3: 调整游戏参数

#### 📏 蛇初始长度
- 拖动滑块：3 ←→ 10
- 推荐值：4（平衡）

#### ⚡ 移动速度
- 拖动滑块：100 ←→ 500 px/s
- 推荐值：200（标准）

#### 🔲 单元格大小
- 拖动滑块：30 ←→ 60 px
- 推荐值：40（适中）

### Step 4: 设置分数

修改不同类型食物的得分：

```
🍎 普通食物：10 分     (范围：1-100)
⭐ 奖励食物：50 分     (范围：10-200)
💎 特殊食物：100 分    (范围：50-500)
```

### Step 5: 管理组件

控制游戏功能的加载：

| 组件 | 作用 | 建议 |
|------|------|------|
| ✨ **粒子效果** | 吃食物、碰撞特效 | 性能优先可关闭 |
| ▦ **网格渲染** | 显示网格线 | 简洁模式可关闭 |
| 🖼️ **背景渲染** | 游戏背景 | 建议保持开启 |
| ⏸️ **暂停管理** | ESC/空格暂停 | 建议保持开启 |

**操作**: 点击右侧开关切换 ✅/❌

### Step 6: 高级选项

#### 🔄 动态难度调整
- **启用**: 根据得分自动提升难度
- **禁用**: 固定当前难度
- **推荐**: 启用（更有挑战性）

#### ⏸️ 自动暂停（失焦）
- **启用**: 切屏时自动暂停
- **禁用**: 不自动暂停
- **推荐**: 启用（防止偷跑）

#### ✨ 粒子效果
- **启用**: 显示所有特效
- **禁用**: 无粒子效果
- **推荐**: 低配设备可禁用

### Step 7: 应用配置

调整完成后，点击底部按钮：

```
🔄 恢复默认   ✅ 应用配置
```

- **恢复默认**: 重置为出厂设置
- **应用配置**: 保存并关闭弹窗

## 🎯 常见问题

### Q1: 配置什么时候生效？

**答**: 配置会在**下次启动游戏**时生效。如果已经在游戏中，需要重新开始一局。

### Q2: 配置会丢失吗？

**答**: 不会！配置保存在浏览器的本地存储中，除非清除浏览器数据，否则会一直保留。

### Q3: 可以创建多个配置方案吗？

**答**: 当前版本只支持保存一个配置。未来版本可能会增加配置预设功能。

### Q4: 如何恢复到默认配置？

**答**: 在配置弹窗中点击"🔄 恢复默认"按钮即可。

### Q5: 关闭粒子效果会影响游戏体验吗？

**答**: 粒子效果是纯视觉特效，关闭后不会影响游戏的核心玩法，反而可能提升性能和 FPS。

## 🎁 进阶技巧

### 技巧 1: 逐步提升难度

不要一开始就选择最高难度，建议：
1. 从"简单"开始熟悉游戏
2. 适应后提升到"普通"
3. 技术成熟后尝试"困难"
4. 最后挑战"极限"

### 技巧 2: 自定义手感

每个人的喜好不同，可以尝试：
- **慢速大格**: 速度 150，单元格 50（容易控制）
- **快速小格**: 速度 400，单元格 30（更具挑战）
- **标准均衡**: 速度 200，单元格 40（比赛标准）

### 技巧 3: 性能优化

如果游戏卡顿，可以：
1. ❌ 关闭粒子效果（+20% FPS）
2. ❌ 关闭网格渲染（+10% FPS）
3. ✅ 保持背景渲染（影响不大）

### 技巧 4: 专注训练

关闭不必要的视觉干扰：
1. ❌ 关闭粒子效果（减少干扰）
2. ✅ 开启网格渲染（帮助定位）
3. ✅ 开启自动暂停（防止意外）

## 🔄 配置重置

### 方法 1: 通过 UI 重置

1. 点击"游戏配置"按钮
2. 点击"🔄 恢复默认"
3. 点击"✅ 应用配置"

### 方法 2: 清除本地存储

打开浏览器开发者工具：
```javascript
// 清除贪吃蛇配置
localStorage.removeItem('snake_game_config')

// 或者清除所有数据
localStorage.clear()
```

然后刷新页面即可。
