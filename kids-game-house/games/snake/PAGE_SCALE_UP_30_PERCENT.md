## 🔧 优化方案

### 响应式 UI 系统

页面使用了 `useResponsiveUI()` 系统，通过 `getWidth()` 和 `getFontSize()` 实现响应式尺寸。

**核心思路**: 在现有计算基础上 × 1.3

---

### 具体调整项

#### 1. 返回按钮 (+30%)

**修改前**:
```typescript
const backBtnStyle = computed(() => ({
  width: ui.getWidth(44),
  height: ui.getWidth(44),
  minWidth: ui.getWidth(44),
}))
```

**修改后**:
```typescript
const backBtnStyle = computed(() => ({
  width: ui.getWidth(44 * 1.3),  // ⭐ 放大 30%
  height: ui.getWidth(44 * 1.3),
  minWidth: ui.getWidth(44 * 1.3),
}))
```

**实际尺寸变化**:
- 44px → **57.2px** (桌面端)

---

---

#### 5. 开始游戏按钮 (+30%)

**修改前**:
```vue

  :fontSize="26"
  :paddingLeft="36"
  :paddingRight="36"
  :paddingTop="18"
  :paddingBottom="18"
/>
```

**修改后**:
```vue
<GameButton
  :fontSize="Math.round(26 * 1.3)"  <!-- ⭐ 33.8px -->
  :paddingLeft="Math.round(36 * 1.3)"  <!-- ⭐ 46.8px -->
  :paddingRight="Math.round(36 * 1.3)"
  :paddingTop="Math.round(18 * 1.3)"  <!-- ⭐ 23.4px -->
  :paddingBottom="Math.round(18 * 1.3)"
/>
```

**实际尺寸变化**:
- 字体：26px → **34px**
- 水平内边距：36px → **47px**
- 垂直内边距：18px → **23px**
