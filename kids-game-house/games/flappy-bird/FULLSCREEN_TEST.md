## 🐛 已知问题和限制

### 1. 浏览器安全策略
**问题：** 某些情况下浏览器会阻止全屏 API 调用

**原因：**
- 需要用户交互上下文
- 异步操作可能失去上下文
- 浏览器扩展可能阻止

**解决方案：**
- ✅ 使用 F11 键（绕过 API 限制）
- ✅ 添加错误提示
- ✅ 提供备选方案

### 2. 多显示器环境
**问题：** 在某些多显示器 setup 下，全屏可能只在主显示器工作

**解决方案：**
- 建议在主显示器运行游戏
- 使用 F11 而非 API

### 3. 远程桌面
**问题：** 远程桌面环境下全屏可能受限

**解决方案：**
- 使用本地显示器
- 或使用 F11 浏览器全屏

## 🎯 最终答案

**问：PC 横向不能全屏吗？**

**答：可以全屏！推荐使用 F11 键。**

1. **F11 键** - 最可靠的方式 ⭐⭐⭐⭐⭐
   - 适用于所有浏览器
   - 横屏竖屏都支持
   - 不受安全策略限制
   - 双显示器也能用

2. **全屏按钮** - 备选方案 ⭐⭐⭐⭐
   - 依赖浏览器 Fullscreen API
   - 可能被安全策略阻止
   - 某些环境不支持

3. **ESC 键** - 退出全屏 ⭐⭐⭐⭐⭐
   - 随时可用
   - 系统级快捷键
