# 🎨 背景滚动修复 - 快速指南

## ❌ 问题现象

背景滚动时出现断层，无法形成无缝循环。

---

## ✅ 解决方案（双图循环）

### 核心代码

```typescript
// 创建两张相同的背景图
const bg1 = this.add.image(0, 0, 'bg_main').setOrigin(0)
const bg2 = this.add.image(0, -this.screenH, 'bg_main').setOrigin(0)

// 同时向下滚动
this.tweens.add({
  targets: [bg1, bg2],
  y: this.screenH,
  duration: 8000,  // 8 秒滚一个屏幕
  ease: 'Linear',
  repeat: -1,
  onUpdate: () => {
    // 滚出屏幕时瞬间移回顶部
    if (bg1.y >= this.screenH) bg1.y = -this.screenH
    if (bg2.y >= this.screenH) bg2.y = -this.screenH
  }
})
```

---

## 🎯 工作原理

### 视觉示意

```
初始状态：          滚动中：            交换位置：
┌─────────┐        ┌─────────┐         ┌─────────┐
│  bg1(0) │        │  bg1(y) │         │  bg2(续)│
├─────────┤   →    ├─────────┤    →    ├─────────┤
│ bg2(-H) │        │ bg2(y-H)│         │bg1(-H 移)│
└─────────┘        └─────────┘         └─────────┘
用户看到 bg1        平滑向下滚动        bg2 接替，bg1 瞬移
```

### 关键点

1. **双图交替**：永远有一张在屏幕上
2. **同步滚动**：同速移动
3. **隐形瞬移**：在屏幕外交换
4. **无缝衔接**：无视觉跳跃

## ⚙️ 参数调整

### 滚动速度

```typescript
duration: 8000,  // 8 秒/圈
```

- 更慢：`12000`（12 秒）
- 更快：`5000`（5 秒）

### 图片要求

✅ 上下边缘可衔接（星空、云层）  
✅ 高度 ≥ 屏幕高度（建议 1080px+）  
✅ 无明显边界线

---

## ✨ 验证方法

刷新浏览器，观察背景：

**应该看到**：
- ✅ 平滑的向下滚动
- ✅ 无任何跳跃或断层
- ✅ 持续循环不断裂

**不应该看到**：
- ❌ 突然的向上跳跃
- ❌ 明显的边界线
- ❌ 滚动不连贯
