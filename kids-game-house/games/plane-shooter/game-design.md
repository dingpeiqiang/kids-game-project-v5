## 一、游戏概述

### 1.1 游戏名称
- **中文名**: 飞机大战
- **英文名**: Plane Shooter
- **游戏代码**: `plane-shooter`

### 1.2 游戏简介
玩家控制一架战斗机，在太空中与敌方战机进行战斗。通过移动躲避敌人子弹并消灭敌机，获得积分。支持三种难度级别，适合不同年龄段的儿童。

### 1.3 核心玩法
- **玩家控制**: 左右移动战斗机
- **自动射击**: 玩家战机会自动发射子弹
- **敌人生成**: 敌机从屏幕上方不断出现
- **碰撞检测**: 
  - 玩家子弹击中敌机 → 敌机被击毁，玩家得分
  - 敌机撞击玩家 → 玩家生命数减 1
  - 敌机子弹击中玩家 → 玩家生命数减 1
- **游戏结束**: 玩家生命数归零时游戏结束

## 二、游戏规则

### 2.1 玩家系统
- **初始生命数**: 3
- **移动方式**: 左右滑动或点击屏幕两侧
- **射击方式**: 自动连续射击
- **得分规则**: 
  - 击毁普通敌机：+10 分
  - 击毁精英敌机：+20 分
  - 击毁 BOSS 敌机：+50 分

### 2.2 敌机系统
- **普通敌机**: 
  - 速度较慢
  - 不发射子弹
  - 直接撞击玩家
- **精英敌机**: 
  - 速度中等
  - 会发射子弹
  - 需要多次击中才能摧毁
- **BOSS 敌机**: 
  - 速度缓慢
  - 大量子弹
  - 需要多次击中才能摧毁
  - 每 30 秒出现一次

### 2.3 道具系统 (可选扩展)
- **生命恢复**: 恢复 1 点生命
- **火力增强**: 子弹数量翻倍
- **护盾**: 抵挡一次伤害
- **炸弹**: 清除屏幕上所有敌人

### 2.4 难度设置
- **简单模式**: 
  - 敌机生成速度慢
  - 敌机子弹少
  - 玩家生命 +1
- **普通模式**: 
  - 标准敌机生成速度
  - 标准敌机子弹密度
- **困难模式**: 
  - 敌机生成速度快
  - 敌机子弹密集
  - 玩家生命 -1

## 三、游戏流程

### 3.1 开始界面
1. 显示游戏标题"飞机大战"
2. 显示游戏说明
3. "开始游戏"按钮
4. 选择难度级别

### 3.2 游戏进行中
1. 玩家战斗机出现在屏幕底部中央
2. 敌机从顶部随机位置生成并向下移动
3. 玩家自动射击
4. 实时显示分数和生命数
5. 根据难度生成不同类型的敌机

### 3.3 游戏结束
1. 显示最终得分
2. 根据得分给予星级评价
   - ⭐⭐⭐⭐⭐: 1000 分以上
   - ⭐⭐⭐⭐: 800-999 分
   - ⭐⭐⭐: 500-799 分
   - ⭐⭐: 300-499 分
   - ⭐: 100-299 分
3. "再来一局"按钮
4. "返回主页"按钮
