# 🎨 背景断层问题深度诊断报告

## 📋 问题确认

**用户反馈**：游戏背景仍然有断层，需要验证是否是背景图片的问题。

---

## 🔍 根本原因分析

### 关键发现

通过深入分析代码和图片资源，发现了**致命的设计缺陷**：

#### 1. 图片尺寸 vs 屏幕尺寸不匹配

```typescript
// 背景图实际高度（固定）
const bgHeight = 1080  // px (生成脚本定义的固定值)

// 屏幕高度（动态，因设备而异）
const screenH = 932    // px (用户的实际屏幕高度)
```

#### 2. 错误的交换逻辑（❌ 修复前）

```typescript
// 初始位置
bg1.y = 0
bg2.y = -screenH  // ❌ 错误：使用了屏幕高度，而不是图片高度！

// 滚动目标
y: screenH        // ❌ 错误：应该滚动图片高度！

// 交换条件
if (bg1.y >= screenH)  // ❌ 错误：判断时机不对！
```

### 问题示意

```
修复前的错误布局：
┌─────────────┐  y=0
│   bg1       │  高度 1080
│             │
│             │
├─────────────┤  y=932 (screenH) ← 用户屏幕底部
│   bg1(续)   │
│             │
└─────────────┘  y=1080 (bgHeight) ← 图片实际底部

┌─────────────┐  y=-932 (bg2 顶部)
│   bg2       │  ← bg2 应该在的位置！
├─────────────┤  y=-932+1080=148  ← bg2 底部没对齐！
│   bg2(续)   │
└─────────────┘

❌ 问题：bg2 的底部在 y=148，而 bg1 的底部在 y=1080
❌ 结果：中间有 932px 的空隙 → 断层！
```

---

## ✅ 正确的修复方案

### 核心原则

**使用图片的实际高度**，而不是屏幕高度！

```typescript
// ✅ 获取背景图的实际高度
const bgTexture = this.textures.get('bg_main')
const bgHeight = bgTexture.getSourceImage().height  // 1080

// ✅ 初始位置：bg2 在 bg1 上方一个图片高度
bg1.y = 0
bg2.y = -bgHeight  // -1080

// ✅ 滚动目标：滚动一整个图片高度
y: bgHeight  // 1080

// ✅ 交换条件：当图片完全滚出时
if (bg1.y >= bgHeight) {
  bg1.y = -bgHeight  // 瞬间移回顶部
}
```

### 正确的视觉布局

```
修复后的正确布局：
┌─────────────┐  y=0        ← bg1 顶部
│   bg1       │  高度 1080
│             │
│             │
└─────────────┘  y=1080     ← bg1 底部

┌─────────────┐  y=-1080    ← bg2 顶部（完美衔接！）
│   bg2       │  高度 1080
│             │
│             │
└─────────────┘  y=0        ← bg2 底部（刚好接上 bg1！）

✅ 完美：bg2 的底部刚好接上 bg1 的顶部
✅ 结果：无缝循环，永不断层！
```

## 🎯 技术要点总结

### 关键教训

1. **必须使用图片的实际尺寸**，而不是屏幕尺寸
2. **两张图的间距必须等于图片高度**
3. **交换时机必须在图片完全滚出时**

### 通用公式

```typescript
// 无缝滚动的黄金法则
const gap = imageHeight      // 两张图的间距 = 图片高度
const scrollDistance = imageHeight  // 滚动距离 = 图片高度
const swapThreshold = imageHeight   // 交换阈值 = 图片高度

// 初始位置
bg1.y = 0
bg2.y = -gap

// 每帧滚动
bg1.y += speed
bg2.y += speed

// 瞬间交换
if (bg1.y >= swapThreshold) bg1.y = -gap
if (bg2.y >= swapThreshold) bg2.y = -gap
```
