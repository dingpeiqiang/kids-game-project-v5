### 输入框样式

#### 优化前 ❌
```css
input {
  margin-bottom: 60px;    /* 固定边距 */
  margin-top: 180px;      /* 不灵活 */
  margin-left: 90px;      /* 不居中 */
  height: 50px;
  font-size: 32px;        /* 过大 */
  width: 300px;
  /* 无边框、无圆角、无阴影 */
}
```

**问题：**
- ❌ 使用固定像素边距，不响应式
- ❌ 不居中显示
- ❌ 字体过大
- ❌ 无视觉反馈
- ❌ 无交互状态

#### 优化后 ✅
```css
input {
  margin: 20px auto;              /* 自动居中 */
  padding: 12px 20px;             /* 合理内边距 */
  height: 50px;
  font-size: 24px;                /* 适中字号 */
  width: 320px;
  border: 2px solid #4a90e2;      /* 蓝色边框 */
  border-radius: 8px;             /* 圆角 */
  background-color: rgba(255, 255, 255, 0.95);  /* 半透明白 */
  color: #333;                    /* 深灰文字 */
  text-align: center;             /* 文字居中 */
  transition: all 0.3s ease;      /* 平滑过渡 */
  box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);     /* 阴影 */
}

input:focus {
  outline: none;
  border-color: #357abd;
  box-shadow: 0 6px 12px rgba(74, 144, 226, 0.3);
  transform: translateY(-2px);    /* 上移动画 */
}
```

**改进：**
- ✅ 自动居中对齐
- ✅ 合理的内边距和外边距
- ✅ 适中的字体大小
- ✅ 美观的边框和圆角
- ✅ 聚焦时的视觉反馈
- ✅ 平滑的交互动画

---

### 按钮样式

#### 优化前 ❌
```css
button {
  border: none;
  background: #1ba0de;            /* 纯色 */
  color: white;
  font-size: 32px;                /* 过大 */
  padding: 7px 5px;               /* 过小 */
  margin-left: 130px;             /* 不居中 */
  border-radius: 5px;
}

button:hover {
  cursor: pointer;                /* 仅改变光标 */
}
```

**问题：**
- ❌ 纯色背景，缺乏层次
- ❌ 字体过大
- ❌ 内边距过小，点击区域小
- ❌ 不居中
- ❌ 悬停效果单一

#### 优化后 ✅
```css
button {
  border: none;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);  /* 渐变 */
  color: white;
  font-size: 24px;                /* 适中 */
  font-weight: bold;              /* 加粗 */
  padding: 12px 32px;             /* 充足内边距 */
  margin: 15px auto;              /* 居中 */
  border-radius: 8px;
  cursor: pointer;
  transition: all 0.3s ease;
  box-shadow: 0 4px 15px rgba(102, 126, 234, 0.4);
  display: block;
  min-width: 180px;
}

button:hover {
  background: linear-gradient(135deg, #764ba2 0%, #667eea 100%);  /* 反向渐变 */
  transform: translateY(-3px);    /* 上移 */
  box-shadow: 0 6px 20px rgba(102, 126, 234, 0.6);  /* 增强阴影 */
}

button:active {
  transform: translateY(-1px);    /* 轻微回弹 */
  box-shadow: 0 3px 10px rgba(102, 126, 234, 0.4);
}

button:disabled {
  background: #ccc;
  cursor: not-allowed;
  box-shadow: none;
  transform: none;
}
```

**改进：**
- ✅ 渐变背景，更有质感
- ✅ 适中的字体大小
- ✅ 充足的点击区域
- ✅ 自动居中对齐
- ✅ 丰富的交互状态（悬停、点击、禁用）
- ✅ 平滑的动画效果
