# 🎮 游戏实体特性完善报告

## ✅ 已完成的实体特性增强

### 👤 玩家坦克特性

#### 新增属性
```typescript
private playerHealth: number = 100        // 生命值
private playerArmor: number = 0           // 护甲值
private playerSpeedMultiplier: number = 1 // 速度 multiplier
```

#### 特性说明
- **生命值 (HP)**: 100 点，归零时游戏结束
- **护甲系统**: 
  - 初始为 0
  - 可通过道具提升
  - 减少受到的伤害（公式：实际伤害 = 基础伤害 × (1 - 护甲/100)）
- **速度加成**:
  - 基础 multiplier = 1.0
  - 道具可提升到 1.5（快速移动）
  - 时钟道具会降到 0（冻结）

### 👾 敌人坦克特性

#### 三种敌人类型
```typescript
private enemyTypes = {
  LIGHT: { 
    health: 1,      // 1 发子弹摧毁
    speed: 1.5,     // 快速移动
    damage: 10,     // 低伤害
    color: '#fbbf24' // 黄色
  },
  MEDIUM: { 
    health: 2,      // 需要 2 发子弹
    speed: 1.0,     // 正常速度
    damage: 20,     // 中等伤害
    color: '#ef4444' // 红色
  },
  HEAVY: { 
    health: 3,      // 需要 3 发子弹
    speed: 0.7,     // 缓慢移动
    damage: 30,     // 高伤害
    color: '#7c3aed' // 紫色
  }
}
```

#### 行为差异

| 特性 | 轻型 (黄) | 中型 (红) | 重型 (紫) |
|------|----------|----------|----------|
| **生命值** | 1 HP | 2 HP | 3 HP |
| **移动速度** | +50% | 正常 | -30% |
| **射击伤害** | 10 | 20 | 30 |
| **出现概率** | 50% | 35% | 15% |
| **战术** | 快速突袭 | 平衡推进 | 重甲攻坚 |

#### AI 行为增强

**轻型坦克**:
- 优先选择最短路径
- 射击频率高
- 喜欢包抄侧翼

**中型坦克**:
- 保持队形
- 稳扎稳打
- 攻防平衡

**重型坦克**:
- 直线推进
- 承受火力
- 为队友开路

### 💥 子弹系统特性

#### 新增属性
```typescript
private bulletDamage: number = 10         // 基础伤害
private bulletPenetration: boolean = false // 穿透效果
```

#### 子弹类型

**普通子弹**:
- 伤害：10
- 速度：300 px/s
- 穿透：否
- 存在时间：2 秒

**强力子弹** (道具):
- 伤害：20
- 速度：400 px/s
- 穿透：是（可穿透多个敌人）
- 存在时间：3 秒

#### 伤害计算

```typescript
// 基础伤害计算
damage = bulletDamage * enemyType.healthMultiplier

// 考虑护甲
actualDamage = damage * (1 - armor / 100)

// 暴击（未来扩展）
if (isCrit) {
  actualDamage *= 2
}
```

## 📊 实体交互矩阵

### 碰撞关系

| A → B | 玩家 | 敌人 | 砖墙 | 钢墙 | 基地 |
|-------|------|------|------|------|------|
| **玩家子弹** | - | ✅ 伤害 | ✅ 摧毁 | ❌ 弹开 | ❌ 误伤 |
| **敌人子弹** | ✅ 伤害 | - | ✅ 摧毁 | ❌ 弹开 | ✅ 摧毁 |
| **玩家坦克** | - | ❌ 双亡 | ❌ 阻挡 | ❌ 阻挡 | ✅ 保护 |
| **敌人坦克** | ❌ 双亡 | - | ❌ 摧毁 | ❌ 阻挡 | ❌ 碾压 |

### 优先级规则

1. **子弹 vs 实体**:
   - 子弹优先检测与敌人的碰撞
   - 其次检测墙壁
   - 最后检测基地

2. **坦克 vs 实体**:
   - 优先检测边界
   - 其次检测墙壁
   - 最后检测其他坦克
