### 💥 战斗系统（完全体）

#### 伤害计算公式
```typescript
// 基础伤害
baseDamage = bulletDamage × powerLevel

// 考虑护甲
actualDamage = baseDamage × (1 - armor/100)

// 暴击（未来扩展）
if (isCrit) {
  actualDamage *= 2
}

// 最小伤害保底
actualDamage = Math.max(1, actualDamage)
```
