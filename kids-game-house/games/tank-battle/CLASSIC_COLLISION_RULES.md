# 🎮 经典坦克大战碰撞规则实现

## 📋 经典游戏规则

### 1. 玩家坦克碰撞规则

| 碰撞对象 | 效果 | 说明 |
|---------|------|------|
| **敌人坦克** | 物理阻挡 | 玩家不能穿过敌人坦克，会被挡住 |
| **敌人子弹** | 受伤/死亡 | 扣除生命或进入死亡动画 |
| **墙壁** | 物理阻挡 | 玩家不能穿过墙壁 |
| **道具** | 拾取效果 | 获得相应能力提升 |
| **基地** | 无效果 | 玩家可以经过基地旁边 |

### 2. 敌人坦克碰撞规则

| 碰撞对象 | 效果 | 说明 |
|---------|------|------|
| **玩家坦克** | 物理阻挡 | 敌人不能穿过玩家，继续移动 |
| **其他敌人** | 物理阻挡 | 敌人之间互相阻挡 |
| **墙壁** | 物理阻挡 + AI 转向 | 遇到墙壁自动改变方向 |
| **玩家子弹** | 受伤/死亡 | 根据子弹威力决定 |
| **基地** | 游戏结束 | 敌人碰到基地则游戏结束 |

### 3. 子弹碰撞规则

#### 玩家子弹
| 碰撞对象 | 效果 | 说明 |
|---------|------|------|
| **敌人坦克** | 造成伤害 | 根据伤害值决定是否需要多次击中 |
| **砖墙** | 摧毁墙壁 | 子弹消失，墙壁被破坏 |
| **钢墙** | 子弹反弹 | 产生火花，墙壁无损 |
| **敌人子弹** | 互相抵消 | 两颗子弹都销毁 |
| **基地** | 无效果 | 玩家子弹不会伤害基地 |

#### 敌人子弹
| 碰撞对象 | 效果 | 说明 |
|---------|------|------|
| **玩家坦克** | 造成伤害 | 玩家扣命或死亡 |
| **砖墙** | 摧毁墙壁 | 子弹消失，墙壁被破坏 |
| **钢墙** | 子弹反弹 | 产生火花，墙壁无损 |
| **玩家子弹** | 互相抵消 | 两颗子弹都销毁 |
| **基地** | 游戏结束 | 基地被摧毁，游戏结束 |

## 🔧 技术实现方案

### 核心原则

1. **物理碰撞使用 `collider`** - 阻止移动
2. **伤害检测使用 `overlap`** - 不阻止移动，只检测接触
3. **无敌状态检查** - 在伤害回调中优先判断
4. **护盾优先拦截** - 护盾 > 无敌 > 护甲 > 生命

### 实现代码

```typescript
// 1. 玩家 vs 敌人坦克（物理阻挡）
physics.add.collider(player, enemies)

// 2. 敌人子弹 vs 玩家（伤害检测）
physics.add.overlap(enemyBullets, player, (bullet, playerObj) => {
  if (!player.invincible) {
    player.takeDamage()
    bullet.destroy()
  }
})

// 3. 玩家子弹 vs 敌人（伤害检测）
physics.add.overlap(playerBullets, enemies, (bullet, enemy) => {
  enemy.health -= bullet.damage
  if (enemy.health <= 0) {
    enemy.destroy()
    addScore(enemy.score)
  }
  bullet.destroy()
})

// 4. 子弹 vs 墙壁
physics.add.collider(bullets, walls, (bullet, wall) => {
  if (wall.type === 'brick') {
    wall.destroy()
    bullet.destroy()
  } else if (wall.type === 'steel') {
    bullet.destroy()
    // 播放火花特效
  }
})

// 5. 敌人子弹 vs 基地
physics.add.overlap(enemyBullets, base, () => {
  gameOver()
})
```

## ⚠️ 常见错误与修复

### 错误 1：玩家被撞击后无法移动
**原因**：手动设置击退速度与每帧速度重置冲突
**修复**：移除手动 setVelocity，依赖物理引擎自动处理

### 错误 2：透明度恢复失败
**原因**：定时器最后一轮回调覆盖可见性设置
**修复**：显式调用 timer.remove(false) 防止最后回调

### 错误 3：敌人穿过玩家
**原因**：缺少 collider 设置
**修复**：添加 physics.add.collider(enemies, player)

## 📊 性能优化

1. **对象池** - 子弹、爆炸特效使用对象池
2. **碰撞过滤** -  inactive 的实体不参与碰撞
3. **距离检测** - 先做平方距离检测，再做精确碰撞
4. **分组管理** - 不同类型的实体使用不同的组
