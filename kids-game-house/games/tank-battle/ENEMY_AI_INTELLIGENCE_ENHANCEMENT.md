## 增强功能总览

### 1. 智能射击系统

#### 原始逻辑
- ❌ 子弹追踪玩家（朝玩家方向发射）
- ❌ 不考虑炮口朝向

#### 增强后逻辑
- ✅ 子弹沿炮口方向发射
- ✅ 判断玩家是否在前方
- ✅ 前方有玩家时正常射击
- ✅ 重型敌人会转向朝向玩家后射击
- ✅ 轻型/中型敌人等待转向后再射击

#### 实现方法
```typescript
// 判断玩家是否在敌人前方
const playerAhead = this.isPlayerAhead(enemy, player, velocity)

if (playerAhead) {
  // 正常射击
  shouldShoot = true
} else if (enemy.enemyType === 'ENEMY_HEAVY') {
  // 重型敌人转向射击
  this.turnTowardsPlayer(enemy, player)
  shouldShoot = true
}
```

### 2. 智能移动系统

#### 原始逻辑
- ❌ 随机选择安全方向
- ❌ 不考虑玩家位置
- ❌ 所有敌人行为一致

#### 增强后逻辑
- ✅ 评分系统：每个方向根据多个因素计算得分
- ✅ 优先选择能接近玩家的方向
- ✅ 根据敌人类型调整策略
- ✅ 加入随机性，避免行为完全一致

#### 评分因素

1. **安全性得分**：基础 100 分（不会撞墙）
2. **追逐得分**：距离玩家越近，得分越高
3. **类型特性得分**：
   - 轻型敌人：更激进，优先追逐（额外 +20% 追逐得分）
   - 重型敌人：更谨慎，增加随机性（-30 分随机）

#### 代码实现
```typescript
// 计算方向得分
let score = 100  // 基础安全性得分

// 追逐得分：距离玩家越近，得分越高
if (distanceToPlayer < currentDistance) {
  score += (currentDistance - distanceToPlayer) * 10
}

// 轻型敌人：更激进
if (enemy.enemyType === 'ENEMY_LIGHT') {
  score += (currentDistance - distanceToPlayer) * 20
}
```

### 3. 分级 AI 系统

#### 轻型敌人（ENEMY_LIGHT）
- **速度**：150（最快）
- **生命值**：1
- **AI 特性**：
  - 移动间隔：800-2000ms（最频繁）
  - 射击间隔：1500-3000ms（最频繁）
  - 追逐策略：激进，优先快速接近玩家
  - 转向行为：不转向射击，等待转向

#### 中型敌人（ENEMY_MEDIUM）
- **速度**：100（中等）
- **生命值**：2
- **AI 特性**：
  - 移动间隔：1000-3000ms
  - 射击间隔：2000-4000ms
  - 追逐策略：平衡
  - 转向行为：不转向射击，等待转向

#### 重型敌人（ENEMY_HEAVY）
- **速度**：50（最慢）
- **生命值**：3
- **AI 特性**：
  - 移动间隔：1500-3500ms（最少）
  - 射击间隔：2500-5000ms
  - 追逐策略：谨慎，增加随机性
  - 转向行为：**智能转向朝向玩家后射击**

## 新增方法

### 1. `isPlayerAhead(enemy, player, velocity)`
判断玩家是否在敌人前方

**参数**：
- `enemy`: 敌人坦克对象
- `player`: 玩家坦克对象
- `velocity`: 敌人当前速度

**返回**：
- `true`: 玩家在前方
- `false`: 玩家不在前方

**逻辑**：
- 根据敌人的移动方向判断
- 允许 100px 的容差范围

---

### 2. `turnTowardsPlayer(enemy, player)`
转向朝向玩家（重型敌人专属）

**参数**：
- `enemy`: 敌人坦克对象
- `player`: 玩家坦克对象

**行为**：
- 计算玩家相对于敌人的主要方向（水平或垂直）
- 调整敌人移动方向朝向玩家
- 更新敌人炮口朝向和纹理

**用途**：
- 重型敌人射击前智能转向
- 提高射击命中率
