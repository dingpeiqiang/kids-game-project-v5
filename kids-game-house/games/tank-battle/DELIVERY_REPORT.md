## 🎯 安装运行说明

### 环境要求
- Node.js >= 18.x
- npm >= 9.x
- 现代浏览器（Chrome/Firefox/Edge）

### 快速启动

```bash
# 1. 进入项目目录
cd kids-game-house/games/tank-battle

# 2. 安装依赖
npm install

# 3. 生成资源文件
npm run generate:resources

# 4. 启动开发服务器
npm run dev
```

浏览器会自动打开 http://localhost:5175

## 📞 技术支持

### 常见问题排查

**问题 1: 游戏无法启动**
```bash
# 解决方案
npm install --force
rm -rf node_modules package-lock.json
npm install
```

**问题 2: 资源加载失败**
```bash
# 重新生成资源
npm run generate:resources
```

**问题 3: 端口被占用**
```bash
# 修改 vite.config.ts 中的 port 配置
server: {
  port: 5176  // 改为其他端口
}
```
