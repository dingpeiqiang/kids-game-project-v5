## 🎮 关卡机制详解

### 关卡过渡

#### 进入新关卡时
```
1. 显示关卡信息（控制台）
2. 重置所有敌人和子弹
3. 保留玩家属性和火力等级
4. 重新创建地图
5. 恢复玩家位置
6. 开始生成新敌人
```

#### 保留的内容
- ✅ 玩家生命值
- ✅ 玩家护甲值
- ✅ 火力等级（1-3 级）
- ✅ 已收集的道具效果
- ✅ 当前得分

#### 重置的内容
- ❌ 敌人位置和状态
- ❌ 所有子弹
- ❌ 未拾取的道具
- ❌ 计时器
- ❌ 生成定时器

---

### 关卡完成检测

```typescript
// 每次击毁敌人时检测
destroyEnemy(enemy) {
  enemy.destroy()
  this.score += 100
  
  // 检查是否所有敌人都被消灭
  this.checkLevelComplete()
}

checkLevelComplete() {
  if (this.enemies.countActive(true) === 0) {
    console.log(`✅ 第${this.currentLevel}关完成！`)
    
    // 延迟 2 秒后进入下一关
    this.time.delayedCall(2000, () => {
      this.currentLevel++
      
      if (this.currentLevel > this.totalLevels) {
        this.handleVictory()  // 通关游戏
      } else {
        this.loadLevel(this.currentLevel)  // 加载下一关
      }
    })
  }
}
```
