## 🎨 视觉效果增强


### 2. 果汁喷射增强

```typescript
// 粒子数量随连击数倍增
const comboMultiplier = 1 + (this.combo * 0.3)
```

### 1. 粒子管理

- 自动清理过期粒子（life <= 0）
- 限制最大粒子数量
- 使用对象池减少GC压力

- requestAnimationFrame 确保60fps
- 只在必要时重绘
- 屏幕震动使用transform而非重绘背景

### 3. 碰撞检测
