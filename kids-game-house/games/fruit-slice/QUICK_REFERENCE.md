# 🍉 水果切切 - 快速参考卡

## 🎮 核心玩法
```
滑动切割水果 → 获得分数 → 避开炸弹 → 完成10关
```

---

| 项目 | 说明 |

| **外观** | 💣 黑色炸弹图标 |

| **惩罚** | -50分 + 连击归零 |

## 🎯 计分公式
```
得分 = 10 × 连击数 × 关卡倍数 × 道具倍数
```

**示例**:
- 第3关，5连击: `10 × 5 × 3 = 150分`
- 第7关，8连击+双倍: `10 × 8 × 7 × 2 = 1120分`

## 💡 高分技巧

1. ✅ **保持连击**: 连续切割获得倍数奖励
2. ✅ **识别炸弹**: 黑色💣=危险，避开！
3. ✅ **优先水果**: 先切水果再考虑连击
4. ✅ **利用道具**: 双倍分数时机很重要
5. ✅ **稳定节奏**: 不要贪快，准确度优先

---

## ⚠️ 常见错误

❌ 切割所有物体（包括炸弹）  
❌ 追求速度忽略准确性  
❌ 不看提示盲目切割  
❌ 炸弹出现时慌乱
