# 🍉 Fruit Slice - 水果切切游戏

一个令人满足的水果切割游戏，具有逼真的果汁喷射效果、沉浸式音效和刺激的炸弹挑战！

**🎉 v2.0 更新**: 新增10关卡系统和炸弹机制！

## ✨ 特性

### 🎮 核心玩法
- **流畅的切割体验**：快速滑动手指或鼠标切割飞起的水果
- **10关卡系统**：从新手到高手的渐进式难度提升
- **💣 炸弹机制**：从第2关开始出现炸弹，考验你的反应和判断
- **连击系统**：连续切割触发连击加成，获得更高分数
- **道具系统**：收集和使用特殊道具增强游戏体验

### 🎨 视觉效果优化
- **果汁喷射粒子**：切割水果时喷射出逼真的果汁粒子效果
- **闪光特效**：切割轨迹带有闪光和光晕效果
- **渐变背景**：深色渐变背景突出游戏元素
- **水果旋转动画**：水果在空中自然旋转飞行

### 🔊 音效系统
- **切割音效**：清脆的撕裂声，根据连击数变化音调
- **果汁喷射音**：湿润的溅射声音效果
- **连击音效**：随连击数升高的华丽音效
- **道具音效**：每种道具都有独特的音效反馈
- **胜利音效**：游戏结束时的欢快旋律

### 🎯 道具系统
- **🐌 减速**：所有水果速度减半，持续8秒
- **🧲 磁铁**：水果向屏幕中心聚集，持续6秒
- **⭐ 双倍分数**：10秒内分数翻倍
- **💣 炸弹**：消除屏幕上所有水果

- **道具使用**：点击底部道具栏使用道具

## 🎯 游戏技巧

### 基础技巧
1. **保持连击**：快速连续切割以获得分数加成
2. **合理使用道具**：在关键时刻使用道具最大化收益
3. **注意时间**：每关有限时，尽量多切割水果
4. **避免漏掉**：不要让水果飞出屏幕

### 💣 炸弹应对策略
5. **识别炸弹**：黑色💣图标=危险，千万不要切！
6. **谨慎选择**：后期近一半是炸弹，需要快速判断
7. **接受惩罚**：不小心切到炸弹会扣50分并重置连击
8. **保持稳定**：不要因为炸弹而慌乱，保持节奏

### 高分策略
9. **优先水果**：先确保切割水果，再考虑连击
10. **利用倍数**：高关卡的倍数奖励非常可观
11. **道具时机**：双倍分数在高连击时使用效果最佳
12. **练习反应**：后期速度极快，需要肌肉记忆

## 🎨 优化亮点

### 1. 果汁喷射效果
- 使用多种粒子类型（果汁、闪光、切割）
- 物理模拟（重力、速度衰减）
- 颜色和大小随机化增加真实感

### 2. 音效系统
- Web Audio API 程序化生成，无需外部音频文件
- 动态音调调整（连击时音调升高）
- 多层音效叠加创造丰富听感

### 3. 性能优化
- 对象池管理粒子和水果
- 高效的碰撞检测算法
- 请求AnimationFrame确保流畅动画
