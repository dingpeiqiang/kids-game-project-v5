# 🍉 Fruit Slice - 水果切切游戏

一个令人满足的水果切割游戏，具有逼真的果汁喷射效果和沉浸式音效。

## ✨ 特性

### 🎮 核心玩法
- **流畅的切割体验**：快速滑动手指或鼠标切割飞起的水果
- **连击系统**：连续切割触发连击加成，获得更高分数
- **道具系统**：收集和使用特殊道具增强游戏体验

### 🎨 视觉效果优化
- **果汁喷射粒子**：切割水果时喷射出逼真的果汁粒子效果
- **闪光特效**：切割轨迹带有闪光和光晕效果
- **渐变背景**：深色渐变背景突出游戏元素
- **水果旋转动画**：水果在空中自然旋转飞行

### 🔊 音效系统
- **切割音效**：清脆的撕裂声，根据连击数变化音调
- **果汁喷射音**：湿润的溅射声音效果
- **连击音效**：随连击数升高的华丽音效
- **道具音效**：每种道具都有独特的音效反馈
- **胜利音效**：游戏结束时的欢快旋律

### 🎯 道具系统
- **🐌 减速**：所有水果速度减半，持续8秒
- **🧲 磁铁**：水果向屏幕中心聚集，持续6秒
- **⭐ 双倍分数**：10秒内分数翻倍
- **💣 炸弹**：消除屏幕上所有水果

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

游戏将在 http://localhost:3010 自动打开

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 🎮 游戏控制

- **鼠标/触摸**：滑动切割水果
- **连击**：快速连续切割多个水果触发连击
- **道具使用**：点击底部道具栏使用道具

## 🎯 游戏技巧

1. **保持连击**：快速连续切割以获得分数加成
2. **合理使用道具**：在关键时刻使用道具最大化收益
3. **注意时间**：游戏限时60秒，尽量多切割水果
4. **避免漏掉**：不要让水果飞出屏幕

## 🛠️ 技术栈

- **TypeScript**：类型安全的代码
- **Canvas API**：高性能2D渲染
- **Web Audio API**：程序化音效生成
- **Vite**：快速的开发服务器和构建工具

## 🎨 优化亮点

### 1. 果汁喷射效果
- 使用多种粒子类型（果汁、闪光、切割）
- 物理模拟（重力、速度衰减）
- 颜色和大小随机化增加真实感

### 2. 音效系统
- Web Audio API 程序化生成，无需外部音频文件
- 动态音调调整（连击时音调升高）
- 多层音效叠加创造丰富听感

### 3. 性能优化
- 对象池管理粒子和水果
- 高效的碰撞检测算法
- 请求AnimationFrame确保流畅动画
