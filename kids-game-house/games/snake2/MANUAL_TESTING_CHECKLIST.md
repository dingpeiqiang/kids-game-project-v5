### 步骤 3: 性能测试

#### FPS 测试

打开浏览器控制台 (F12)，输入：

```javascript
// FPS 计数器
let frameCount = 0;
let lastTime = performance.now();

setInterval(() => {
  frameCount++;
  const now = performance.now();
  if (now - lastTime >= 1000) {
    console.log(`📊 FPS: ${frameCount}`);
    frameCount = 0;
    lastTime = now;
  }
}, 1000);
```

**标准**: 
- ✅ 稳定 60 FPS
- ⚠️ 50-60 FPS
- ❌ < 50 FPS

---

#### 内存测试

在 Chrome 控制台：

```javascript
// 查看内存占用
console.log(`💾 内存占用：${(performance.memory?.usedJSHeapSize / 1048576).toFixed(2)} MB`);
```

**标准**:
- ✅ < 120 MB
- ⚠️ 120-150 MB
- ❌ > 150 MB

**标准**:

## 🎯 自动化测试（可选）

### E2E 测试脚本

使用 Playwright 进行端到端测试：

```bash
# 安装 Playwright
npm install -D @playwright/test

# 初始化配置
npx playwright init
```

创建测试文件 `tests/e2e/game.spec.ts`:

```typescript
import { test, expect } from '@playwright/test';

test('游戏正常启动', async ({ page }) => {
  await page.goto('http://localhost:5173/');
  
  // 等待游戏加载
  await expect(page.locator('.game-canvas')).toBeVisible({ timeout: 10000 });
  
  // 检查分数面板
  await expect(page.locator('.score-panel')).toBeVisible();
});

test('蛇能够正常移动', async ({ page }) => {
  await page.goto('http://localhost:5173/');
  
  // 等待游戏开始
  await page.waitForTimeout(2000);
  
  // 按方向键
  await page.keyboard.press('ArrowRight');
  await page.waitForTimeout(500);
  await page.keyboard.press('ArrowDown');
  
  // 验证游戏仍在运行
  await expect(page.locator('.game-canvas')).toBeVisible();
});
```

运行测试：

```bash
npx playwright test
```
