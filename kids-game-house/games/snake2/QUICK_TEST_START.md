## ⚡ 超简单三步测试

### 第 1 步：打开 PowerShell（Windows）或 Terminal（Mac/Linux）

在项目中打开终端：
```bash
# 在项目根目录右键 → "在终端中打开"
# 或使用快捷键 Ctrl+` (Windows)
```

---

### 第 2 步：运行以下命令

```powershell
# 进入贪吃蛇项目目录
cd kids-game-house\games\snake

# 安装依赖（如果已安装可跳过）
npm install

# 启动开发服务器（配置端口：3005）
npm run dev
```

**看到以下输出表示成功**：
```
  VITE v5.0.0  ready in 500 ms

  ➜  Local:   http://localhost:3005/
  ➜  Network: use --host to expose
  ➜  press h to show help
```

---

### 第 3 步：打开浏览器访问

```
http://localhost:3005/
```

**注意**: 
- 如果 3005 端口被占用，Vite 会自动使用 3006、3007 等
- 查看终端输出中的实际端口号

## 💡 常见问题速查

### Q1: 页面打不开？

**解决方案**:
```bash
# 检查端口是否被占用
netstat -ano | findstr :3005

# 如果端口被占用，可以：
# 方法 1: 杀掉占用进程
# 方法 2: 换用其他端口
npm run dev -- --port 3006
```

---

### Q2: 黑屏/白屏？

**检查步骤**:
1. 按 F12 打开开发者工具
2. 查看 Console 标签
3. 查找红色错误信息
4. 根据错误信息排查

**常见原因**:
- TypeScript 编译错误
- 资源加载失败
- 浏览器兼容性问题

---

### Q3: 按键没反应？

**检查步骤**:
1. 确认游戏窗口获得焦点（点击游戏区域）
2. 检查键盘是否正常
3. 查看 Console 是否有错误

---

### Q4: 很卡顿怎么办？

**优化建议**:
1. 关闭其他占用资源的程序
2. 降低浏览器硬件加速
3. 清理浏览器缓存
