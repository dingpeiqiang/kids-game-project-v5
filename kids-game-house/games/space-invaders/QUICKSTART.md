# Space Invaders - 快速开始指南

## 🎮 立即开始游戏

### Windows 用户
双击运行 `start.bat` 文件

### 所有用户
```bash
npm run dev
```

然后在浏览器打开: **http://localhost:8080**

---

## 🕹️ 游戏控制

| 按键 | 功能 |
|------|------|
| ← → | 左右移动飞船 |
| 空格 | 发射子弹 |
| ESC | 暂停/继续 |
| R | 重新开始(游戏结束后) |

---

## 🎯 游戏目标

1. **消灭所有外星人** - 进入下一关
2. **保护生命** - 你有3条命
3. **阻止入侵** - 别让外星人到达底部
4. **挑战高分** - 等级越高,分数越多

---

## ✨ 离线版特性

- ✅ 无需网络即可运行
- ✅ 无需 API 密钥
- ✅ 所有资源本地生成
- ✅ 多关卡系统
- ✅ 生命系统 (3条命)
- ✅ 道具系统 (护盾/速射/额外生命)
- ✅ 音效系统 (Web Audio API)
- ✅ 粒子特效 (华丽爆炸)
- ✅ 最高分记录 (本地存储)
- ✅ 暂停功能 (ESC键)
- ✅ 流畅动画效果
