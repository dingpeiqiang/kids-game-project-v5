# 🎮 实时多人太空侵略者游戏

## ⚡ 快速启动

```bash
npm run dev
```

访问 http://localhost:8080 开始游戏！

## 🔑 重要提示

首次运行前需要：
1. 复制 `.env-sample` 为 `.env`
2. 在 `.env` 中填入你的 Ably API 密钥
3. 运行 `npm install` 安装依赖

获取 Ably API 密钥：https://ably.com/sign-up

## 🛠️ 可用命令

| 命令 | 说明 |
|------|------|
| `npm run dev` | 开发模式（自动重载）✨ |
| `npm start` | 生产模式 |
| `dev.bat` | Windows 一键启动 |
| `./dev.sh` | Mac/Linux 一键启动 |
