## ✨ 全新功能 (8项)

### 1. 🎯 连击系统 (Combo System)

**功能**:
- 连续击杀累积连击数
- 2秒超时重置
- 连击倍率增加分数 (每连击+10%)
- 实时显示连击数

**视觉反馈**:
- 右上角显示 "5x COMBO!"
- 5连击: 橙色
- 10连击+: 金色闪烁

**成就**:
- 🏆 连击大师 (5连击)
- 🏆 连击之神 (10连击)

---

### 2. 🔫 敌人射击系统

**功能**:
- 敌人随机向下发射红色子弹
- 玩家需躲避攻击
- 难度影响射击频率

**难度平衡**:
- Easy: 3秒间隔
- Normal: 2秒间隔
- Hard: 1秒间隔

**战术意义**:
- 增加挑战性
- 需要走位躲避
- 不能无脑站桩输出

---

### 3. 👹 Boss战系统

**功能**:
- 每5关出现Boss (Level 5, 10, 15...)
- Boss拥有高生命值 (20 + Level×2)
- Boss会三发散射子弹
- Boss定期左右移动

**Boss特性**:
- 紫色大型飞船
- 绿色血条显示
- 1.5秒射击间隔
- 2秒移动一次

**奖励**:
- 分数: Level × 500
- 必掉高级道具
- 成就: Boss杀手

## 🎮 游戏玩法

### 基本操作
```
← →     : 左右移动
SPACE   : 发射子弹
ESC     : 暂停/继续
TAB     : 显示/隐藏统计
R       : 重新开始 (结束后)
```

### 游戏流程

1. **选择难度** → 根据技能选择
2. **输入昵称** → 个性化体验
3. **开始战斗** → 消灭敌人
4. **收集道具** → 获得增益
5. **达成连击** → 分数飙升
6. **躲避射击** → 生存关键
7. **挑战Boss** → 每5关
8. **解锁成就** → 长期目标
9. **打破纪录** → 竞争最高分

## 🎯 游戏技巧

### 高分策略

1. **保持连击**
   - 快速连续击杀
   - 不要中断超过2秒
   - 连击倍率显著增加分数

2. **优先收集道具**
   - 护盾: 应对密集弹幕
   - 速射: 快速清场+高连击
   - 生命: 延长生存时间

3. **精准射击**
   - 提高命中率 (>80%解锁成就)
   - 避免浪费子弹
   - 瞄准移动预判

4. **Boss战技巧**
   - 观察射击模式
   - 利用间隙攻击
   - 保持移动躲避

### 难度建议

**新手**:
- 从Easy开始
- 熟悉基本操作
- 练习躲避技巧
- 逐步提升难度

**进阶**:
- Normal标准体验
- 追求高连击
- 挑战全成就
- 优化命中率

**高手**:
- Hard极限挑战
- 无伤通关
- 速通挑战
- 冲击最高分

## 🏆 成就指南

### 快速解锁路线

1. **首次击杀** → 消灭第一个敌人 (立即解锁)
2. **连击大师** → 使用速射道具快速击杀5个
3. **连击之神** → 同上,保持10连击
4. **分数破千** → 正常游戏即可达成
5. **进阶玩家** → 通过前4关
6. **Boss杀手** → 到达第5关击败Boss
7. **高分玩家** → 多玩几局,使用道具
8. **资深战士** → 通过前9关
9. **神射手** → 保持命中率>80%
10. **完美通关** → 无伤通过任意一关

### 最难成就

**神射手** (命中率>80%):
- 需要精准射击
- 避免乱开枪
- 建议在Easy模式练习
- 使用统计面板监控

**完美通关**:
- 不能受到任何伤害
- 建议使用护盾道具
- 小心敌人射击
- 保持安全距离
