# 🎮 游戏模式选择指南

## 快速对比

| | 🕹️ 单机模式 | 👥 多人模式 |
|---|---|---|
| **API 密钥** | ❌ 不需要 | ✅ 需要 Ably |
| **启动命令** | `npm run single` | `npm run dev` |
| **玩家数量** | 1人 | 2-6人 |
| **配置难度** | ⭐ 零配置 | ⭐⭐⭐ 需注册 |
| **适合场景** | 个人练习、测试 | 朋友联机 |

## 🕹️ 单机模式（推荐新手）

### ✅ 优点
- **即时启动** - 无需任何配置
- **无需注册** - 不需要创建账户
- **离线可用** - 完全本地运行
- **快速测试** - 适合开发和调试

### ❌ 限制
- 只能单人游戏
- 无法与朋友联机

### 🚀 启动方法
```bash
cd realtime-multiplayer-space-invaders-main
npm run single
```

访问：http://localhost:8080

---

## 👥 多人模式（完整体验）

### ✅ 优点
- **实时联机** - 最多6人同时游戏
- **社交互动** - 与朋友一起游玩
- **房间系统** - 创建/加入游戏房间
- **完整功能** - 所有原始特性

### ❌ 限制
- 需要 Ably API 密钥
- 需要注册账户
- 配置稍复杂

### 🔑 获取 API 密钥步骤

1. 访问 https://ably.com/sign-up
2. 注册免费账户
3. 创建新应用
4. 复制 API 密钥
5. 编辑 `.env` 文件填入密钥

### 🚀 启动方法
```bash
cd realtime-multiplayer-space-invaders-main
npm install
npm run dev
```

访问：http://localhost:8080

---

## 💡 建议

### 选择单机模式，如果：
- ✅ 您想立即开始游戏
- ✅ 您只是想测试一下
- ✅ 您不想注册任何账户
- ✅ 您是开发者，想快速查看效果

### 选择多人模式，如果：
- ✅ 您想和朋友一起玩
- ✅ 您需要完整的多人游戏体验
- ✅ 您愿意花几分钟注册账户
- ✅ 您要部署到生产环境

---

## 🎯 我的推荐

**第一次尝试？** → 使用 **单机模式** (`npm run single`)
- 零配置，立即游玩
- 体验游戏核心玩法

**想和朋友玩？** → 使用 **多人模式** (`npm run dev`)
- 需要 Ably API 密钥
- 完整的多人在此体验
