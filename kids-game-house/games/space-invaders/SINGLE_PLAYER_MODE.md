# 🎮 单机模式 - 无需 API 密钥即可运行！

## ✅ 好消息！

您**不需要** Ably API 密钥就可以运行游戏了！我们创建了一个**单机模式**版本。

## 🚀 快速启动（推荐）

### 方法一：使用启动脚本
```bash
start-single.bat
```

### 方法二：使用 npm 命令
```bash
npm run single
```

### 方法三：开发模式（带自动重载）
```bash
npm run dev:single
```

然后打开浏览器访问：**http://localhost:8080**

## 🎯 单机模式特点

- ✅ **无需 API 密钥** - 完全本地运行
- ✅ **即时启动** - 不需要注册任何账户
- ✅ **完整游戏体验** - 经典的太空侵略者玩法
- ✅ **自动重载** - 代码修改后自动刷新（dev:single 模式）

## 🕹️ 游戏控制

- **← → 方向键** - 移动飞船
- **空格键 (SPACE)** - 发射子弹
- **目标** - 消灭所有敌人

## 📊 两种模式对比

| 特性 | 单机模式 | 多人模式 |
|------|---------|---------|
| API 密钥 | ❌ 不需要 | ✅ 需要 Ably |
| 玩家数量 | 1人 | 最多6人 |
| 启动难度 | ⭐ 简单 | ⭐⭐⭐ 需配置 |
| 适合场景 | 个人游玩、测试 | 联机对战 |

## 🎮 游戏玩法

1. 启动服务器（`npm run single`）
2. 打开浏览器 http://localhost:8080
3. 使用方向键移动飞船
4. 按空格键射击敌人
5. 消灭所有敌人即可获胜！

## 💡 提示

- 敌人会左右移动并逐渐下降
- 被敌人碰到就会游戏结束
- 尽量躲避敌人的同时消灭它们
- 每消灭一个敌人得 10 分

## 🔧 技术说明

单机模式移除了以下依赖：
- ❌ Ably 实时通信库
- ❌ Worker Threads
- ❌ p2 物理引擎

保留了：
- ✅ Phaser 3 游戏引擎
- ✅ Express 静态文件服务器
- ✅ 完整的游戏逻辑和渲染

## 📝 文件说明

- `server-single.js` - 单机版服务器（无 Ably）
- `script-single.js` - 单机版游戏逻辑
- `index-single.html` - 单机版游戏页面
- `start-single.bat` - Windows 启动脚本

## 🎉 立即开始

```bash
# 在项目目录执行
npm run single

# 或者双击
start-single.bat
```

然后打开浏览器享受游戏吧！🚀
