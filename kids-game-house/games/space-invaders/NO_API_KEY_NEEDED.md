## ❓ 常见问题

### Q: 单机模式和多人模式有什么区别？
A: 单机模式只能一个人玩，但不需要任何配置。多人模式可以和朋友一起玩，但需要 Ably API 密钥。

### Q: 我可以同时运行两种模式吗？
A: 可以，但需要使用不同的端口。修改 `.env` 文件中的 PORT 值即可。

### Q: 单机模式的功能完整吗？
A: 单机模式包含完整的核心游戏玩法，但没有多人联机功能。

### Q: 以后想玩多人模式怎么办？
A: 随时可以切换到多人模式，只需按照 [QUICKSTART.md](QUICKSTART.md) 的说明配置 API 密钥即可。
