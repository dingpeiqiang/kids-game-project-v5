# 🚀 关卡系统快速启动指南

## ⚡ 立即开始

### 1. 启动游戏
```bash
cd kids-game-house/games/car-on-curved-terrain
npm run dev
```

浏览器会自动打开 http://localhost:3000/

---

## 🎮 操作指南

### 基础控制
- **前进**: → 方向键 或 D/W 键
- **后退**: ← 方向键 或 A/S 键
- **也可使用**: 屏幕上的触摸按钮

### 快捷键（新增！）
- **L 键**: 打开关卡选择界面
- **R 键**: 重试当前关卡

## 🎯 游戏玩法

### 基本流程
1. **启动游戏** → 自动进入第1关
2. **驾驶车辆** → 尽可能行驶更远
3. **达到目标** → 获得星级评价
4. **解锁新关** → 挑战更高难度
5. **重复挑战** → 争取3星完美

### 获取高星技巧
- ✅ 保持稳定速度，不要急加速
- ✅ 上坡时提前加速，下坡时适当减速
- ✅ 利用防翻车辅助（倾斜时自动修正）
- ✅ 观察地形，预判坡度变化
- ✅ 多次练习，熟悉每个关卡

## 📊 进度说明

### 自动保存
- ✅ 关卡解锁状态
- ✅ 每关获得的星级
- ✅ 刷新页面不丢失

### 查看位置
进度保存在浏览器 localStorage：
```javascript
localStorage.getItem('carGameLevelProgress')
```

### 重置进度
在浏览器控制台执行：
```javascript
localStorage.removeItem('carGameLevelProgress')
location.reload()
```

## 🐛 常见问题

### Q: 按L键没反应？
A: 确保点击了游戏窗口，让键盘获得焦点

### Q: 如何知道是否完成关卡？
A: 达到目标距离后会自动弹出完成界面

### Q: 可以跳过关卡吗？
A: 不能，必须按顺序完成解锁

### Q: 能重新挑战已完成的关卡吗？
A: 可以！按L键打开关卡选择，点击任意已解锁关卡

### Q: 翻车了怎么办？
A: 按R键快速重试，或等待几秒自动重置
