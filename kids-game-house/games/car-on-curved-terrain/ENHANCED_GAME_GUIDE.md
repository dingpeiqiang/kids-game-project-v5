## 🎯 新增功能一览

### 1. 📊 计分系统
- **实时距离显示** - 看到你行驶了多远
- **最高分记录** - 挑战自己的最佳成绩
- **自动保存** - 使用 localStorage 持久化

### 2. 🏎️ 速度表
- **实时速度** - 以 km/h 显示当前速度
- **档位指示** - N（空档）、D1-D3（前进档）、R（倒档）
- **颜色警示** - 绿色→橙色→红色，直观了解速度

### 3. 💨 粒子效果
- **车轮烟雾** - 加速时产生逼真的烟雾
- **动态反馈** - 根据驾驶状态调整效果
- **性能优化** - 智能发射，不影响帧率

### 4. 🤸 特技系统
- **空中时间追踪** - 看看你能飞多久
- **翻转检测** - 完成360°翻转会被记录
- **统计数据** - 跟踪你的表演技巧

## 🎮 游戏玩法

### 基本控制
- **→ 右箭头** - 加速前进
- **← 左箭头** - 倒车/刹车

### 游戏目标
1. **行驶更远** - 突破你的距离记录
2. **保持平衡** - 避免翻车或掉落
3. **表演特技** - 尝试空中翻转
4. **挑战速度** - 达到最高速度

### 技巧提示
- 🟢 **平稳起步** - 避免突然加速导致翻车
- 🟡 **利用惯性** - 下坡时可以获得更高速度
- 🔴 **注意着陆** - 高空落地可能损坏车辆
- ⭐ **尝试翻转** - 在空中时使用方向键旋转
