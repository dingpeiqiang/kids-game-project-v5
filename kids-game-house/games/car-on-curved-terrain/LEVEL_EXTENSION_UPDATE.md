### 2. 自动进入下一关

#### 功能说明
当玩家完成关卡后：
1. 显示完成界面（星级、距离）
2. **3秒倒计时**开始
3. 倒计时结束后**自动进入下一关**
4. 玩家可以随时点击"重试"或"下一关"按钮中断自动跳转

#### UI展示
```
┌──────────────────────────┐
│   🎉 关卡完成！          │
│   新手之路               │
│                          │
│   行驶距离: 1023m        │
│                          │
│      ⭐  ⭐  ⭐         │
│                          │
│  [🔄 重试]  [➡️ 下一关] │
│                          │
│  3秒后自动进入下一关...  │ ← 新增倒计时
└──────────────────────────┘
```

#### 倒计时逻辑
```typescript
3秒 → "3秒后自动进入下一关..."
2秒 → "2秒后自动进入下一关..."
1秒 → "1秒后自动进入下一关..."
0秒 → "正在跳转..."
     → 自动加载下一关
```

#### 特殊情况处理
- **最后一关**: 不显示倒计时（没有下一关）
- **点击按钮**: 立即中断倒计时，执行对应操作
- **隐藏界面**: 清理定时器，避免内存泄漏

## 🔧 技术实现

### 1. 地形路径扩展

**原理**: SVG贝塞尔曲线路径延长

```typescript
// 之前的路径（较短）
terrain1Path: 'M -10,400 C ... 3500,510'
terrain2Path: 'M 3500,510 C ... 6500,585'

// 现在的路径（更长）
terrain1Path: 'M -10,400 C ... 8300,630'  // 延长了4800px
terrain2Path: 'M 8300,630 C ... 11900,720' // 延长了5400px
```

**关键点**:
- terrain1终点 = terrain2起点（无缝连接）
- 保持平缓的Y轴变化（每200px变化≤15px）
- 桥梁位置与地形高度精确对齐

---

### 2. 自动跳转机制

**核心代码**:
```typescript
// MainScene.ts - 检测关卡完成
const nextLevelId = this.levelManager.unlockNextLevel()

this.levelCompleteUI.show(
  distance,
  stars,
  currentLevel.name,
  () => this.nextLevel(),
  () => this.retryLevel(),
  nextLevelId ? 3 : 0  // 有下一关则3秒后自动跳转
)
```

**LevelCompleteUI.ts - 倒计时逻辑**:
```typescript
private showCountdown(seconds: number, onTimeout: () => void) {
  let remaining = seconds
  
  const updateCountdown = () => {
    if (remaining > 0) {
      this.countdownText!.setText(`${remaining}秒后自动进入下一关...`)
      remaining--
      this.autoNextTimer = this.scene.time.delayedCall(1000, updateCountdown)
    } else {
      this.countdownText!.setText('正在跳转...')
      this.scene.time.delayedCall(500, () => {
        this.hide()
        onTimeout()  // 触发下一关加载
      })
    }
  }
  
  updateCountdown()
}
```

---

### 3. 资源管理

**定时器清理**:
```typescript
// hide() 方法中
if (this.autoNextTimer) {
  this.autoNextTimer.remove()
  this.autoNextTimer = null
}

// destroy() 方法中
if (this.autoNextTimer) {
  this.autoNextTimer.remove()
  this.autoNextTimer = null
}
this.countdownText?.destroy()
```

**防止内存泄漏**:
- ✅ 隐藏界面时清理定时器
- ✅ 销毁UI时清理所有资源
- ✅ 点击按钮时中断倒计时
