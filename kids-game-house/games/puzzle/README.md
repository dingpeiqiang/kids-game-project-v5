# 🧩 快乐拼图屋 - 动物主题

> **适合 3-8 岁儿童的益智拼图游戏** - 通过点击交换拼图块还原完整动物图案，培养观察力和逻辑思维能力。

## 📋 项目信息

- **游戏名称**: 快乐拼图屋 (Happy Puzzle House)
- **游戏类型**: 益智拼图
- **目标用户**: 3-8 岁儿童
- **开发框架**: kids-game-frame-factory v5
- **技术栈**: Phaser 3.90 + Vue 3 + TypeScript
- **主题**: 动物主题（猫咪、小狗、小兔）

## 🎮 核心玩法

### 操作方式
1. **点击选中**: 第一次点击拼图块，选中并高亮显示
2. **再次点击**: 点击另一个拼图块，两者交换位置
3. **胜利条件**: 所有拼图块都回到正确位置时获得胜利
4. **星级评价**: 根据完成时间获得 1-3 星评价

### 难度等级
| 难度 | 网格大小 | 拼图块数 | 适合年龄 | 3 星标准 |
|------|---------|---------|---------|---------|
| **简单 ⭐** | 2×2 | 4 块 | 3-4 岁 | <15 秒 |
| **普通 ⭐⭐** | 3×3 | 9 块 | 5-6 岁 | <30 秒 |
| **困难 ⭐⭐⭐** | 4×4 | 16 块 | 7-8 岁 | <60 秒 |

## 🛠️ 开发指南

### 修改难度参数
编辑 `src/config/difficulty.json`:
```json
{
  "difficulties": [
    {
      "id": "easy",
      "label": "简单 ⭐",
      "description": "2×2 网格，4 块拼图，适合 3-4 岁",
      "gridCols": 2,
      "gridRows": 2,
      "speed": 1000,
      "scoreMultiplier": 1.0
    }
  ]
}
```

### 添加新动物主题
1. 在 `generate-resources.mjs` 中添加新的动物颜色配置
2. 运行 `node generate-resources.mjs` 生成资源
3. 复制生成的 GTRS.json 到 `src/config/`
4. 修改 `MyGameScene.ts` 中的 `animalThemes` 映射

### 修改游戏逻辑
编辑 `src/scenes/MyGameScene.ts`:
- `createGameObjects()`: 创建游戏对象
- `gameLoop()`: 每帧更新逻辑
- `handleGameOver()`: 游戏结束处理
