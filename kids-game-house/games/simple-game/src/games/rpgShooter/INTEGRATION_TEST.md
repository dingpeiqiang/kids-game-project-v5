## 🐛 常见问题排查

### 问题1: 首页看不到游戏卡片

**检查步骤**:
1. 确认 `game-config.ts` 中 `visible: true`
2. 清除浏览器缓存 (Ctrl+Shift+R)
3. 重启开发服务器
4. 检查控制台是否有错误

**解决**:
```bash
# 清除Vite缓存
rm -rf node_modules/.vite
npm run dev
```

### 问题2: 点击卡片无法进入游戏

**检查步骤**:
1. 打开浏览器开发者工具 (F12)
2. 查看Console是否有错误
3. 检查Network是否有资源加载失败
4. 确认 `rpgShooter.ts` 文件存在

**常见错误**:
- 模块导入路径错误
- TypeScript编译错误
- 资源文件缺失

### 问题3: 游戏画面不显示

**检查步骤**:
1. 确认Canvas元素存在
2. 检查Canvas尺寸是否正确
3. 查看是否有绘制错误
4. 确认ctx不为null

**调试代码**:
```javascript
console.log('Canvas:', canvas)
console.log('Context:', ctx)
console.log('Size:', canvas.width, canvas.height)
```

### 问题4: 控制不响应

**检查步骤**:
1. 确认事件监听器已绑定
2. 检查是否有其他元素遮挡
3. 查看Console是否有事件错误
4. 测试鼠标和键盘分别是否正常
