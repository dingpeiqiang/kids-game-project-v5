## 📈 难度递增曲线

### 第1-3关：入门阶段
- **目标**：让玩家熟悉操作
- **特点**：速度慢、饼干少、几乎不旋转
- **适合**：所有年龄段的玩家，特别是小朋友

### 第4-6关：进阶阶段
- **目标**：提升反应速度和手眼协调
- **特点**：速度中等、饼干增多、开始旋转
- **适合**：有一定基础的玩家

### 第7-10关：高手阶段
- **目标**：挑战极限，测试真正实力
- **特点**：速度快、饼干多、旋转剧烈
- **适合**：追求挑战和竞技的玩家

## 🎯 通关机制

### 如何进入下一关？
1. 在当前关卡时间内（60秒）达到目标分数
2. 时间结束后自动检查分数
3. 如果达标且不是最后一关，2秒后自动进入下一关
4. 如果不达标或已是最后一关，游戏结束

### 失败条件
- 时间结束时未达到目标分数
- 主动退出游戏

### 成功条件
- 完成第10关并达到目标分数
- 成为真正的"饼干之神"！🏆

```
关卡1: 垂直速度 2.5, 水平速度 1.0
```

```
关卡5: 旋转速度 0.09（轻微旋转）
```

## 💡 游戏策略

### 新手建议（关卡1-3）
1. **不要着急**：速度慢，有充足时间反应
2. **练习连击**：尝试连续切割获得更高分数
3. **熟悉操作**：掌握滑动切割的节奏

### 进阶技巧（关卡4-6）
1. **预判位置**：饼干开始加速，要提前规划路线
2. **优先连击**：连击倍数能大幅提升分数
3. **使用道具**：合理使用减速、冻结等道具

### 高手秘籍（关卡7-10）
1. **快速反应**：速度极快，需要瞬间决策
2. **精准切割**：一次划过多个饼干
3. **保持冷静**：不要被快节奏打乱节奏
4. **最大化连击**：争取5连击以上的高倍奖励

## 🔧 自定义关卡

如果想调整关卡难度，编辑 `levels.ts` 文件：

```typescript
{
  level: 5,
  name: '切割高手',
  description: ' halfway 了！挑战真正开始...',
  duration: 60000,        // 游戏时长
  spawnInterval: 1400,    // 生成间隔
  maxCookies: 3,          // 最大饼干数
  verticalSpeed: 4.5,     // 垂直速度
  horizontalSpeed: 2.0,   // 水平速度
  targetScore: 700,       // 目标分数
  rotationSpeed: 0.09,    // 旋转速度
}
```

### 调整建议
- **更容易**：降低速度、减少饼干、降低目标分数
- **更困难**：提高速度、增加饼干、提高目标分数
- **更刺激**：缩短时间、加快生成频率
