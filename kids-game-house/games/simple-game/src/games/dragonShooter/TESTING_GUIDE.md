## 🐛 常见问题排查

### 问题1：关卡管理界面不显示
**症状**：点击按钮后没有反应

**排查步骤**：
1. 打开浏览器控制台（F12）
2. 查看是否有 JavaScript 错误
3. 检查 `state.phase` 是否变为 `'routeSelect'`
4. 确认 `drawLevelManager()` 函数存在

**解决方案**：
- 刷新页面
- 清除浏览器缓存
- 检查代码是否有语法错误

### 问题2：关卡列表为空
**症状**：显示 "暂无配置，点击"+"添加"

**排查步骤**：
1. 在控制台输入：`console.log(LEVEL_SPECIFIC_ROUTES)`
2. 检查对象是否为空
3. 确认代码中是否正确定义了配置

**解决方案**：
- 检查 `index.ts` 中的 `LEVEL_SPECIFIC_ROUTES` 定义
- 确保没有被意外删除
- 重新加载页面

### 问题3：点击关卡没有反应
**症状**：点击关卡条目没有浮动文字

**排查步骤**：
1. 检查控制台是否有点击日志
2. 确认点击坐标计算是否正确
3. 检查 `state.floatTexts` 数组是否更新

**解决方案**：
- 检查 `handleDown()` 函数中的点击检测逻辑
- 确认坐标范围计算正确
- 调试点击事件的 x, y 值

### 问题4：路线类型标签颜色不对
**症状**：所有标签都是默认紫色

**排查步骤**：
1. 检查路线 ID 是否包含正确的关键字
2. 确认类型识别逻辑是否正确

**解决方案**：
- 检查 `route.id` 的值
- 确保包含 'wave', 'zigzag', 'spiral', 'boss' 等关键字
- 调试类型识别代码
