# Color Sort 色彩排序游戏

## 📋 概述

经典的液体颜色排序益智游戏，玩家需要将混合的彩色液体分拣到不同的试管中，使每个试管只包含一种颜色。

## 🎮 游戏特色

- ✨ **液体视觉效果**：逼真的液体渲染，包括渐变、高光和表面张力
- 🎯 **10个精心设计的关卡**：从入门到专家的渐进式难度
- 💾 **进度保存**：自动保存游戏进度和最佳成绩
- 🎁 **道具系统**：提示、撤销、额外管子等辅助道具
- 📱 **移动端优化**：完美支持触摸操作

### 难度设计原则

1. **颜色数量递增**：从3色逐步增加到6色
2. **液体层数增加**：从3层增加到7层
3. **时间压力提升**：从180秒减少到60秒
4. **空间复杂度**：管子数量适度增加，提供更多策略空间

## 🎯 游戏规则

1. **目标**：将所有相同颜色的液体分到同一个试管中
2. **操作**：
   - 点击一个有液体的试管选中
   - 点击另一个试管移动液体
   - 只能将液体移动到空管或相同颜色的液体上
3. **胜利条件**：所有试管要么为空，要么装满同色液体
4. **失败条件**：时间耗尽

## 💡 游戏技巧

- **规划先行**：先观察整体布局，制定移动策略
- **保留空管**：空管是宝贵的中转空间，谨慎使用
- **优先完成**：优先完成一个颜色的整理，释放空间
- **避免死局**：不要把所有颜色都分散开
