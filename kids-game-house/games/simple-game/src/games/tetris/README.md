# 🎮 Tetris 俄罗斯方块 - 解压爽快感增强版

## ✨ 优化特性

### 1. **模块化架构**
- ✅ 迁移到独立目录 `tetris/`
- ✅ 采用类结构 `TetrisGame`，代码更清晰
- ✅ 分离粒子系统 `ParticleSystem`，便于维护

### 2. **视觉增强**
- 🌟 **粒子特效系统**
  - 消除行时的火花特效
  - 方块放置的尘埃效果
  - 道具收集的星星爆炸
  - 连击时的绚丽爆炸
  
- 📺 **屏幕震动**
  - Tetris（四行消除）时强烈震动
  - 连击时逐渐增强的震动
  - 提供触觉反馈般的沉浸感

- 🎨 **动态连击显示**
  - 彩虹色连击文字
  - 字体大小随连击数增长
  - 发光效果增强视觉冲击

### 3. **游戏性增强**
- 🔥 **连击系统 (Combo)**
  - 连续消除行累积连击
  - 连击加分：每级连击额外 +10% 分数
  - 3秒无消除自动重置
  
- 💎 **道具系统优化**
  - 🧹 清除一行
  - 💣 Tetris（清除四行）
  - 🐌 减速下落
  - ✨ 双倍分数
  - 👁️ 预览下一个方块

### 4. **音效配合**
- 方块移动：收集音效
- 消除行：胜利音效
- 使用道具：对应音效
- 连击达成：强化音效

## 🚀 技术亮点

### 粒子系统架构
```typescript
class ParticleSystem {
  - createLineClearEffect()  // 消除特效
  - createPlaceEffect()      // 放置特效
  - createPowerupCollectEffect() // 道具特效
  - createComboEffect()      // 连击特效
  - createScreenShake()      // 屏幕震动
  - update()                 // 更新粒子
  - render()                 // 渲染粒子
}
```

### 性能优化
- 粒子自动清理（life <= 0）
- 使用 requestAnimationFrame
- 增量更新（deltaTime）
- 对象池思想（可扩展）
