# Tetris 游戏 - 快速参考指南

## 🚀 快速开始

### 启动游戏
在游戏列表中选择 "Tetris" 即可启动优化后的俄罗斯方块游戏。

### 基本操作
```
键盘控制:
  ← / →  : 左右移动
  ↓      : 加速下落
  ↑      : 旋转方块
  空格    : 直接落底

触摸控制:
  左侧点击  : 左移
  右侧点击  : 右移
  中间点击  : 旋转
```

## 🎯 核心特性

### 1. 连击系统 (Combo)
- 连续消除行累积连击
- 每级连击额外 +10% 分数
- 3秒无消除自动重置
- 连击越高，特效越华丽

**示例:**
```
2连击 → +20% 分数
5连击 → +50% 分数
10连击 → +100% 分数（双倍）
```

### 2. 粒子特效
- **消除行**: 火花四溅
- **放置方块**: 尘埃飞扬
- **收集道具**: 星星爆炸
- **连击达成**: 绚丽爆发

### 3. 屏幕震动
- Tetris (4行消除): 强烈震动
- 连击时: 渐进震动
- 提供沉浸式反馈

### 4. 道具系统
| 道具 | 图标 | 效果 |
|------|------|------|
| 清除一行 | 🧹 | 消除最底下一行 |
| Tetris | 💣 | 消除最底下四行 |
| 减速 | 🐌 | 下落速度减半，持续8秒 |
| 双倍分数 | ✨ | 分数翻倍，持续10秒 |
| 预览 | 👁️ | 高亮显示下一个方块，持续10秒 |

## 📊 计分规则

### 基础分数
```
1行: 100 × 等级
2行: 300 × 等级
3行: 500 × 等级
4行: 800 × 等级
```

### 连击加成
```
最终分数 = 基础分数 × (1 + 连击数 × 0.1)
```

### 特殊加成
```
双倍分数道具: 最终分数 × 2
```

## 🎨 视觉效果

### 连击显示
```
位置: 屏幕中央上方
颜色: 彩虹色渐变 (HSL)
大小: 20px + 连击数 × 2
效果: 发光 + 动态变化
```

### 粒子类型
- **Spark**: 小而快的火花，用于消除行
- **Dust**: 柔和的尘埃，用于放置方块
- **Star**: 五角星形状，用于道具收集
- **Explosion**: 大型爆炸，用于连击

## 💡 游戏技巧

### 高分策略
1. **保持连击**: 尽量连续消除行
2. **规划布局**: 预留空间制造 Tetris
3. **合理使用道具**: 关键时刻使用
4. **快速反应**: 利用加速下落得分

### 连击技巧
- 一次性放置多个方块制造多次消除
- 利用 L/J 形状填补空隙
- 保持场地平整，避免空洞

## 🎮 游戏目标

- **基本目标**: 消除尽可能多的行
- **进阶目标**: 达到高等级 (每10行升一级)
- **挑战目标**: 获得高连击 (10+ 连击)
- **终极目标**: 达到 50 行通关

## ⚠️ 注意事项

1. 方块堆到顶部游戏结束
2. 连击会在3秒无消除后重置
3. 道具库存有限，合理使用
4. 等级越高，下落速度越快
