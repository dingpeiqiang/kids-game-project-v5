### 2. 代码架构升级

#### 从函数式到面向对象
- **之前**: 单一函数 `initTetris()`，所有逻辑混在一起
- **现在**: `TetrisGame` 类，职责清晰，易于维护

#### 模块化设计
- 粒子系统独立封装为 `ParticleSystem` 类
- 游戏逻辑、渲染、输入分离
- 符合单一职责原则

### 3. 视觉增强特性

#### 🌟 粒子特效系统
实现了4种粒子类型：
- **Spark**: 消除行时的火花效果
- **Dust**: 方块放置的尘埃效果
- **Star**: 道具收集的星星爆炸
- **Explosion**: 连击时的绚丽爆炸

每个粒子都有：
- 物理模拟（重力、摩擦力）
- 生命周期管理
- 动态大小和透明度
- 发光效果

#### 📺 屏幕震动
- Tetris（四行消除）时强烈震动（强度8）
- 连击时渐进震动（强度3+连击数）
- 持续时间可配置
- 提供触觉反馈般的沉浸感

#### 🎨 动态连击显示
- 彩虹色渐变文字（HSL色彩空间）
- 字体大小随连击数增长（20px + 连击数×2）
- 发光效果增强视觉冲击
- 居中显示，醒目突出

### 4. 游戏性增强

#### 🔥 连击系统 (Combo)
```typescript
// 连击机制
- 连续消除行累积连击
- 连击加分：基础分数 × (1 + 连击数 × 0.1)
- 3秒无消除自动重置
- 连击越高，分数加成越大
```

**示例：**
- 2连击: +20% 分数
- 5连击: +50% 分数
- 10连击: +100% 分数（双倍）

#### 💎 道具系统优化
保留原有道具系统，并增强视觉效果：
- 🧹 **清除一行**: 金色粒子特效
- 💣 **Tetris**: 彩色粒子爆发 + 屏幕震动
- 🐌 **减速下落**: 持续8秒
- ✨ **双倍分数**: 持续10秒
- 👁️ **预览**: 高亮显示下一个方块

### 5. 性能优化

## 📝 使用说明

### 启动游戏
游戏已通过主索引文件注册，可直接在游戏列表中选择 "Tetris" 启动。

### 操作方式
**键盘控制：**
- ⬅️ 左箭头：左移
- ➡️ 右箭头：右移
- ⬇️ 下箭头：加速下落
- ⬆️ 上箭头：旋转
- 空格键：直接落底

**触摸控制：**
- 左侧区域：左移
- 右侧区域：右移
- 中间区域：旋转

### 道具使用
1. 消除带有道具图标的方块获得道具
2. 道具存储在库存中
3. 点击底部道具栏使用道具
4. 每种道具有不同的效果
