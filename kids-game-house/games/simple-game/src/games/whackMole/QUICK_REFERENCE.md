# 打地鼠游戏 - 快速参考指南

## 🎮 游戏概述

**打地鼠 (Whack-a-Mole)** 是一款Q版卡通风格的解压小游戏，玩家通过点击屏幕击打从洞口探出的地鼠获得分数。

---

## 📋 核心玩法

### 基本规则
- **游戏时长**：默认60秒
- **操作方式**：鼠标点击 / 触屏点击
- **得分机制**：击打探出的地鼠获得分数
- **空击惩罚**：无（降低挫败感）

### 地鼠类型与得分

| 地鼠类型 | 得分 | 出现概率 | 停留时间 | 特效 |
|---------|------|---------|---------|------|
| 🐭 普通地鼠 | +10 | 40-70% | 85帧 | 绿色粒子 |
| ✨ 黄金地鼠 | +30 | 10-15% | 50帧 | 金色闪光 |
| 💥 炸弹地鼠 | -20 | 8-12% | 130帧 | 爆炸特效 |
| 🌟 双倍地鼠 | +15 | 12-18% | 70帧 | 粉色buff |
| 👻 隐身地鼠 | +50 | 0-15% | 35帧 | 紫色粒子 |

## 🎯 难度阶段

### 前期 (0-20秒) - 新手友好
- ⏱️ 刷新间隔：85帧
- 📊 特点：普通地鼠为主，无隐身地鼠
- 🎲 同时出现：单只

### 中期 (20-40秒) - 节奏提升
- ⏱️ 刷新间隔：60帧
- 📊 特点：特殊地鼠增加
- 🎲 同时出现：25%概率双只

### 后期 (40-60秒) - 极速挑战
- ⏱️ 刷新间隔：40帧
- 📊 特点：高难度组合，隐身地鼠高频
- 🎲 同时出现：40%概率双只

## 🎮 特殊机制

### 双倍得分Buff
- **触发条件**：击打双倍地鼠
- **持续时间**：5秒
- **效果**：所有得分×2
- **HUD显示**：实时倒计时
- **自动过期**：时间到自动重置

### 防止重复击打
```typescript
if (m.isHit) return  // 已击打的地鼠不再响应
```

### 漏击无惩罚
- 地鼠自动缩回未被击打
- 无扣分，仅错过得分机会
- 降低玩家挫败感
