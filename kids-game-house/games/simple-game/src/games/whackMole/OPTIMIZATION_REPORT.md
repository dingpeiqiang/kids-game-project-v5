# 打地鼠游戏优化报告

## 📋 优化概览

本次优化基于用户需求，对打地鼠游戏进行了全面升级，包括新增特殊地鼠类型、优化难度梯度、增强音效震动反馈、改进美术风格等。

---

## ✨ 核心功能优化

### 1. 新增特殊地鼠类型

#### 1.1 双倍地鼠 (Double Mole)
- **出现概率**：前期12%，中期15%，后期18%
- **得分规则**：基础分+15分，触发5秒双倍得分buff
- **特效**：粉色粒子特效 + ✨标识
- **停留时间**：70帧（中等）
- **视觉**：粉红色调，可爱造型

#### 1.2 隐身地鼠 (Invisible Mole)
- **出现概率**：前期0%，中期8%，后期15%
- **得分规则**：+50分（高分奖励）
- **特效**：紫色粒子特效
- **停留时间**：35帧（极短，考验手速）
- **视觉**：半透明显示（alpha=0.5），高难度挑战

### 2. 优化地鼠类型概率分布

| 地鼠类型 | 前期 (0-20s) | 中期 (20-40s) | 后期 (40-60s) |
|---------|------------|------------|------------|
| 普通地鼠 | 70% | 55% | 40% |
| 黄金地鼠 | 10% | 12% | 15% |
| 炸弹地鼠 | 8% | 10% | 12% |
| 双倍地鼠 | 12% | 15% | 18% |
| 隐身地鼠 | 0% | 8% | 15% |

### 3. 渐进式动态难度系统

#### 前期 (0-20秒) - 新手友好
- 刷新间隔：85帧
- 普通地鼠为主
- 无隐身地鼠
- 单只出现

#### 中期 (20-40秒) - 节奏提升
- 刷新间隔：60帧
- 特殊地鼠增加
- 25%概率双只出现

#### 后期 (40-60秒) - 极速挑战
- 刷新间隔：40帧
- 高难度组合
- 40%概率双只出现
- 隐身地鼠高频出现

## 🎨 美术风格优化

### 1. Q版卡通治愈系设计

#### 地鼠表情系统
- **正常状态**：微笑表情 + 大眼睛 + 高光
- **被击打后**：委屈捂脸表情
  - 闭眼流泪效果
  - 撇嘴表情
  - 眼泪动画（天蓝色）

#### 特殊地鼠视觉
- **黄金地鼠**：金色光圈 + 金光闪烁特效
- **炸弹地鼠**：黑色球体 + ☠符号 + 导火索火花
- **双倍地鼠**：粉红色调 + 星星特效
- **隐身地鼠**：半透明显示（50%透明度）

#### 星空 (Starry)
- 渐变：深蓝夜空 → 深棕地面
- 风格：梦幻浪漫

#### 海岛 (Seaside)
- 渐变：天蓝天空 → 浅蓝海水 → 沙滩
- 风格：夏日清凉

### 3. UI优化
- 扁平化极简设计
- 圆润柔和按钮
- 清晰可爱字体
- 无冗余元素

## 🎮  gameplay 优化

### 1. 分数系统
- **普通地鼠**：+10分
- **黄金地鼠**：+30分
- **双倍地鼠**：+15分（触发5秒双倍buff）
- **隐身地鼠**：+50分
- **炸弹误击**：-20分（降低惩罚，减少挫败感）

### 2. 双倍得分Buff
- 持续时间：5秒
- HUD实时显示剩余时间
- 自动过期重置
- 适用于所有得分（包括普通、黄金、隐身）

### 3. 防止重复击打
```typescript
if (m.isHit) return  // 已击打的地鼠不再响应
```

### 4. 空击无惩罚
- 点击无地鼠洞口：无得分，无惩罚
- 点击已缩回地鼠：无反应
- 降低玩家挫败感

## 📊 技术实现细节

### 1. 数据结构扩展

```typescript
type MoleType = 'normal' | 'gold' | 'bomb' | 'double' | 'invisible'

interface Mole {
  holeIdx: number
  type: MoleType
  state: 'rising' | 'visible' | 'hiding' | 'idle' | 'hit'
  offsetY: number
  timer: number
  showDuration: number
  hitScale: number
  hitAlpha: number
  isHit?: boolean  // 新增：防止重复击打
}
```

### 2. 游戏状态管理

```typescript
let scoreMultiplier = 1          // 分数倍数
let doubleScoreEndTime = 0       // 双倍buff结束时间
let currentPhase: 'early' | 'mid' | 'late' = 'early'  // 难度阶段
let backgroundTheme = 'grassland' // 背景主题
```

### 3. 新增特效函数

- `emitExplosionEffect()` - 炸弹爆炸特效
- `emitBuffEffect()` - 双倍buff特效
- `emitGoldEffect()` - 黄金地鼠金光特效

### 4. 绘制函数增强

```typescript
drawMoleBody(c, x, y, w, h, isGold, isInvisible, isHit)
```
- 支持隐身地鼠半透明渲染
- 支持被击打后的委屈表情
- 动态切换睁眼/闭眼状态
