# Bubble Shooter 泡泡龙游戏

## 🎮 游戏简介

泡泡龙是一款经典的休闲益智游戏，玩家通过发射彩色泡泡来消除同色泡泡，获得分数并完成关卡挑战。

## 🎯 游戏玩法

### 基本操作
- **鼠标/触摸控制**：移动鼠标或手指来控制发射器方向
- **点击/触摸发射**：点击屏幕或松开手指发射泡泡
- **消除规则**：3个或更多同色泡泡相连时自动消除

### 游戏目标
- 消除棋盘上所有泡泡即可过关
- 在时间限制内完成关卡获得更高分数
- 通关所有10个关卡成为泡泡龙大师！

## ✨ 特殊泡泡

游戏中会随机出现特殊泡泡，增加游戏趣味性和策略性：

| 图标 | 名称 | 效果 |
|------|------|------|
| 🌈 | 彩虹泡泡 | 可以匹配任何颜色的泡泡，消除时自动选择最优匹配 |
| 💣 | 炸弹泡泡 | 消除时产生爆炸，摧毁周围3x3范围内的所有泡泡 |
| 🟡 | 粘性泡泡 | 附着后不会因浮动而掉落，增加策略深度 |

## 🎁 道具系统

发射泡泡时有概率获得道具，道具会在下次发射时生效：

| 图标 | 名称 | 效果 |
|------|------|------|
| 💣 | 颜色炸弹 | 随机消除场上所有同色泡泡 |
| 🧹 | 清除行 | 消除最底部一行的所有泡泡 |
| ⚡ | 额外射击 | 额外获得一次发射机会 |
| 🔫 | 三连射 | 连续发射三个泡泡 |
| ⏸️ | 时间冻结 | 暂停时间5秒，策略思考的好时机 |
| 🐢 | 慢动作 | 泡泡飞行速度减慢3倍，更容易瞄准 |
| 💎 | 双倍分数 | 8秒内获得双倍分数加成 |

## 🔥 连击系统

连续消除泡泡可触发连击奖励：

- **连击条件**：3秒内连续消除泡泡
- **最大倍数**：5倍分数加成
- **视觉反馈**：
  - 3连击：显示"3 连击!"（红色）
  - 5连击：显示"超爽!"（粉色）
- **额外奖励**：3连击以上触发随机增益效果

## 📊 分数计算

- **基础分数**：每个消除的泡泡20分
- **连击加成**：连击倍数 × 基础分数
- **浮动奖励**：消除后掉落的泡泡每个30分
- **道具奖励**：使用道具获得额外分数
- **时间奖励**：关卡剩余时间转化为奖励分数

## 🚀 游戏技巧

1. **瞄准技巧**：利用墙壁反弹可以到达难以直接瞄准的位置
2. **策略消除**：优先消除底部泡泡，可让上方泡泡掉落获得额外分数
3. **道具利用**：合理使用道具可以快速清理棋盘
4. **特殊泡泡**：优先消除炸弹泡泡引发连锁爆炸
5. **连击追求**：快速连续消除获得高分加成
