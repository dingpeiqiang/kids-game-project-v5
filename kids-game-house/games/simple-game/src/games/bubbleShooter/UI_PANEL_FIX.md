# Bubble Shooter UI 面板优化说明

## 🎯 问题描述

**用户反馈**: "游戏信息面板不对，遮挡了游戏画面"

**问题分析**:
- UI 元素（分数、连击、时间）直接绘制在 Canvas 底部
- 没有背景面板，文字与游戏内容重叠
- 遮挡了发射器和飞行中的泡泡
- 可读性差，视觉效果混乱

## ✅ 解决方案

为所有 UI 元素添加**半透明黑色背景面板**，确保：
1. UI 信息清晰可读
2. 不遮挡游戏画面关键元素
3. 视觉层次分明
4. 符合 Canvas UI 设计规范

## 🔧 实施细节

### 1. 分数显示优化

#### 修改前 ❌
```typescript
// 直接绘制文字，无背景
ctx.fillText(String(score), centerX, this.H - 15)
```

#### 修改后 ✅
```typescript
// 先绘制半透明背景面板
ctx.fillStyle = 'rgba(0, 0, 0, 0.6)'
ctx.fillRect(centerX - 60, this.H - 45, 120, 40)

// 再绘制文字
ctx.fillText(String(score), centerX, this.H - 15)
```

**效果**:
- 背景面板: 120px 宽 × 40px 高
- 位置: 居中，距离底部 15px
- 透明度: 60% 黑色，既突出文字又不完全遮挡

### 2. 连击显示优化

#### 修改前 ❌
```typescript
// 直接绘制连击文字
ctx.fillText(`${combo} 连击!`, centerX, this.H - 115)
```

#### 修改后 ✅
```typescript
// 先绘制连击背景面板
ctx.fillStyle = 'rgba(0, 0, 0, 0.6)'
ctx.fillRect(centerX - 50, this.H - 140, 100, 35)

// 再绘制连击文字
ctx.fillText(`${combo} 连击!`, centerX, this.H - 115)
```

**效果**:
- 背景面板: 100px 宽 × 35px 高
- 位置: 居中，距离底部 115px
- 只在连击 ≥ 2 时显示

### 3. 时间显示优化

#### 修改前 ❌
```typescript
// 直接绘制时间文字
ctx.fillText(`${seconds}s`, this.W - 10, this.H - 15)
```

#### 修改后 ✅
```typescript
// 动态计算文字宽度
const timeText = `${seconds}s`
const timeWidth = ctx.measureText(timeText).width

// 根据文字宽度绘制背景面板
ctx.fillStyle = 'rgba(0, 0, 0, 0.6)'
ctx.fillRect(this.W - timeWidth - 20, this.H - 45, timeWidth + 20, 40)

// 再绘制时间文字
ctx.fillText(timeText, this.W - 10, this.H - 15)
```

**效果**:
- 背景面板: 自适应宽度（文字宽度 + 20px 边距）
- 位置: 右对齐，距离右边 10px
- 最后 10 秒变红色警告

## 🎨 设计规范

### 背景面板样式
```typescript
// 统一的半透明黑色背景
ctx.fillStyle = 'rgba(0, 0, 0, 0.6)'
```

**选择 60% 透明度的原因**:
- ✅ 足够暗，确保白色/彩色文字清晰可见
- ✅ 足够透明，不完全遮挡背后的游戏内容
- ✅ 视觉上轻盈，不显得笨重
- ✅ 符合现代 UI 设计趋势

### 面板尺寸原则

| UI 元素 | 宽度 | 高度 | 边距 | 说明 |
|---------|------|------|------|------|
| 分数 | 120px | 40px | 左右各 10px | 固定宽度，容纳最多 6 位数字 |
| 连击 | 100px | 35px | 左右各 10px | 固定宽度，适配常见连击数 |
| 时间 | 自适应 | 40px | 左右各 10px | 根据文字动态调整 |

## 💡 技术要点

### 1. 动态宽度计算

时间显示使用 `measureText()` 动态计算宽度：

```typescript
const timeText = `${seconds}s`
const timeWidth = ctx.measureText(timeText).width
```

**优势**:
- 适配不同长度的时间（"99s" vs "5s"）
- 背景面板始终紧贴文字
- 避免浪费空间或截断文字

### 2. 像素对齐

所有坐标都使用 `Math.round()` 确保清晰：

```typescript
const centerX = Math.round(this.W / 2)
ctx.fillRect(centerX - 60, this.H - 45, 120, 40)
```

### 3. 渲染顺序

UI 在最后绘制，确保在最上层：

```typescript
1. 背景
2. 棋盘泡泡
3. 飞行泡泡
4. 发射器
5. 粒子效果
6. UI 信息（最上层）✅
```
