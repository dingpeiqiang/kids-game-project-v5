## 🎮 操作指南

### 建造模式
1. **按B键**进入建造模式
2. **点击标签页**切换炮台/陷阱
3. **点击按钮**选择要建造的类型
4. **移动鼠标**到地图位置
5. **点击左键**放置

### 炮台升级
1. **退出建造模式**（ESC或再按B）
2. **点击地图上的炮台**
3. **查看升级信息**
4. **再次点击**确认升级/出售
5. **按ESC**取消

### 快捷键
- **B** - 进入/退出建造模式
- **1-4** - 快速选择炮台（建造模式下）
- **ESC** - 取消当前操作
