## 🐛 常见问题排查

### 问题1: 敌人不射击
**可能原因**:
- wave参数未正确传递
- ENEMY_SHOOT_CONFIGS配置错误

**解决方法**:
```typescript
// 检查 createEnemy 调用
const enemy = createEnemy(type, x, y, state.difficulty, state.wave)
//                                         ^^^^^^^^^^ 确保传入wave
```

---

### 问题2: 子弹不移动
**可能原因**:
- vx/vy计算错误
- dt值异常

**解决方法**:
```typescript
// 检查 updateEnemyBullets
bullet.x += bullet.vx * 60 * dt
bullet.y += bullet.vy * 60 * dt
//              ^^^^^^^ 确认不为0
```

---

### 问题3: 碰撞检测不准确
**可能原因**:
- 距离计算公式错误
- 半径值不正确

**解决方法**:
```typescript
// 检查碰撞判断
if (distToPlayer < 8 * SCALE_RATIO + bullet.size) {
//                  ^^^^^^^^^^^^^^^ 玩家半径
//                                ^^^^^^^^^^^^ 子弹半径
}
```

---

### 问题4: 性能下降
**可能原因**:
- 子弹未正确清理
- 数组持续增长

**解决方法**:
```typescript
// 确保超出屏幕的子弹被移除
if (bullet.x < -50 || bullet.x > CANVAS_WIDTH + 50 || 
    bullet.y < -50 || bullet.y > CANVAS_HEIGHT + 50) {
  state.enemyBullets.splice(i, 1)  // ← 这行必须执行
  continue
}
```
