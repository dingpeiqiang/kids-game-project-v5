### 1. 虚拟摇杆系统 🕹️

**功能描述**: 
在屏幕左下角添加虚拟摇杆，让玩家可以通过触摸滑动来控制角色移动和瞄准方向。

**实现细节**:

```typescript
// 虚拟摇杆状态
const joystick = {
  active: false,        // 是否激活
  startX: 0,            // 起始X坐标
  startY: 0,            // 起始Y坐标
  currentX: 0,          // 当前X坐标
  currentY: 0,          // 当前Y坐标
  radius: 60,           // 摇杆底座半径
  knobRadius: 25        // 摇杆钮半径
}
```

**激活区域**: 
- 左下角40%宽度 × 底部40%高度区域
- 触摸该区域即可激活虚拟摇杆

**工作原理**:
1. **touchstart**: 检测触摸位置，如果在左下角区域则激活摇杆
2. **touchmove**: 计算手指偏移量，限制在摇杆半径内，更新角色瞄准方向
3. **touchend**: 停止摇杆，角色停止移动

**视觉反馈**:
- **未激活**: 显示提示文字"👆 触摸此处移动"
- **激活中**: 
  - 白色半透明圆圈（底座）
  - 青色半透明圆钮（可拖动）
  - "移动"提示文字

## 🔧 技术实现细节

### 1. 虚拟摇杆算法

```typescript
// 计算手指偏移量
const dx = joystick.currentX - joystick.startX
const dy = joystick.currentY - joystick.startY
const distance = Math.sqrt(dx * dx + dy * dy)

// 限制在摇杆半径内
if (distance > joystick.radius) {
  const ratio = joystick.radius / distance
  joystick.currentX = joystick.startX + dx * ratio
  joystick.currentY = joystick.startY + dy * ratio
}

// 更新瞄准方向
mousePos.x = joystick.currentX
mousePos.y = joystick.currentY
```

**关键点**:
- 使用向量归一化限制移动范围
- 实时更新鼠标位置用于瞄准
- 平滑的视觉反馈

---

### 2. 点击热区扩展算法

```typescript
// 原始点击检测
if (clickX >= btn.x && clickX <= btn.x + btn.w &&
    clickY >= btn.y && clickY <= btn.y + btn.h) {
  // 触发点击
}

// 扩展后的点击检测
const padding = 10
if (clickX >= btn.x - padding && clickX <= btn.x + btn.w + padding &&
    clickY >= btn.y - padding && clickY <= btn.y + btn.h + padding) {
  // 触发点击
}
```

**优势**:
- 容错率提高
- 减少误触失败
- 提升用户体验

---

### 3. 响应式适配

所有UI元素都使用`SCALE_RATIO`进行缩放：

```typescript
const btnSize = 50 * SCALE_RATIO
const padding = 10 * SCALE_RATIO
```

**设备适配**:
- iPhone SE (SCALE_RATIO=0.75): 按钮37.5px, 边距7.5px
- Android小屏 (SCALE_RATIO=0.75): 同上
- Desktop (SCALE_RATIO=1.0): 按钮50px, 边距10px
