## ✅ 优化内容

### 1. Canvas尺寸大幅缩小

**修改历程**：
- v1: 手机端 90%宽度 / 70%高度
- v2: 手机端 85%宽度 / 60%高度  
- **v3: 手机端 75%宽度 / 50%高度** ⬇️⬇️

**代码位置**: `config.ts`

```typescript

const heightPercent = isMobile ? 0.5 : 0.7    // 从60%降至50%
```

---

### 2. 攻击范围再次缩小

**修改历程**：
- v1: 300px（太大）
- v2: 200px → 150px
- **v3: 120px** ⬇️ 再减少20%

**代码位置**: `combat.ts`

```typescript

```

---

### 3. UI按钮缩小

**建造模式按钮**：
- v2: 50x50px
- **v3: 40x40px**（根据缩放比例调整）

**字体大小**：
