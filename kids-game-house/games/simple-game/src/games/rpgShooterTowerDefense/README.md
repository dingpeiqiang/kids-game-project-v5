### 🎮 游戏玩法

1. **开始游戏** - 点击"开始游戏"按钮
2. **控制角色** - WASD/方向键移动，或左下角虚拟摇杆（手机端）
3. **自动射击** - 角色自动瞄准最近的敌人
4. **建造炮台** - 按B键或点击右上角按钮进入建造模式，1-4键选炮台，点击地图放置
5. **出售炮台** - 点击已有炮台可出售（返还50%成本）
6. **抵御8波敌人** - 生存到最后获得胜利！
