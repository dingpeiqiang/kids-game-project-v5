# 📱 手机端操作完整指南

## 🎮 核心操作方式

### 1. 移动角色 - 虚拟摇杆 🕹️

**操作方法**:
- 触摸屏幕**左下角区域**（40%宽度 × 底部40%高度）
- 手指滑动控制移动方向
- 松开手指停止移动

**视觉反馈**:
- 未激活: 显示"👆 触摸此处移动"提示
- 激活中: 白色圆圈底座 + 青色摇杆钮

---

### 2. 切换建造模式 - 双指点击 👆👆

**操作方法**:
- **同时用两根手指点击屏幕任意位置**
- 即可切换建造模式开/关

**视觉反馈**:
- 开启: 底部显示炮台/陷阱选择菜单
- 关闭: 隐藏菜单，返回战斗模式
- 屏幕底部显示提示："双指点击 → 切换建造模式"

**音效**: 播放选择音效

---

### 3. 选择炮台/陷阱 - 点击按钮 🔨

**操作方法**:
- 进入建造模式后
- 点击底部的炮台/陷阱按钮
- 标签页切换：点击"🏰 炮台"或"⚙️ 陷阱"标签

**可用炮台**:
1. **激光炮台** (Laser) - 快速单体攻击
2. **导弹炮台** (Missile) - 追踪爆炸
3. **冰冻炮台** (Frost) - 减速敌人
4. **闪电炮台** (Lightning) - 连锁伤害

**可用陷阱**:
1. **地雷** (Mine) - 💣 30💎
2. **减速力场** (SlowField) - ❄️ 40💎
3. **尖刺陷阱** (Spike) - 🔺 25💎

---

### 4. 放置炮台/陷阱 - 点击地图 📍

**操作方法**:
- 选择炮台/陷阱类型后
- 点击地图上的空位
- 绿色预览 = 可以放置
- 红色预览 = 不能放置

**限制**:
- 不能重叠已有建筑
- 需要足够的水晶资源
- 最多建造数量限制

---

### 5. 升级/出售炮台 - 点击已有炮台 ⬆️

**操作方法**:
- 退出建造模式
- 点击地图上已有的炮台
- 弹出升级/出售对话框

**升级**:
- 显示下一级属性和成本
- 点击确认升级
- 需要足够水晶

**出售**:
- 返还70%建造成本
- 至少保留1个炮台

**取消**: 点击ESC或屏幕外

---

### 6. 开始游戏 - 点击屏幕 ▶️

**操作方法**:
- 游戏启动后
- 点击屏幕任意位置
- 开始第一波敌人

---

## 🎯 操作流程示例

### 典型游戏流程

```
1. 点击屏幕 → 开始游戏
   ↓
2. 左下角触摸 → 移动角色（自动射击）
   ↓
3. 双指点击 → 进入建造模式
   ↓
4. 点击炮台按钮 → 选择激光炮台
   ↓
5. 点击地图 → 放置炮台
   ↓
6. 双指点击 → 退出建造模式
   ↓
7. 继续战斗，重复步骤3-6
   ↓
8. 点击已有炮台 → 升级强化
```

## 💡 操作技巧

### 新手技巧

1. **先建防御**: 第一波前快速建造2-3个炮台
2. **合理布局**: 炮台分散放置，覆盖更多区域
3. **及时升级**: 优先升级主力炮台
4. **保持连击**: 2秒内连续击杀获得奖励

### 进阶技巧

1. **边战边建**: 战斗中也可以建造（双指切换很快）
2. **陷阱配合**: 减速陷阱 + 高伤害炮台 = 完美combo
3. **资源管理**: 不要一次性花光水晶，留有余地
4. **走位躲避**: 利用虚拟摇杆灵活躲避敌人

### 高手技巧

1. **快速切换**: 熟练双指操作，实现无缝建造
2. **精准放置**: 预判敌人路径，提前布置
3. **连击维持**: 保持高连击获得额外奖励
4. **Boss应对**: Boss来临前集中升级关键炮台

## ⚠️ 注意事项

### 常见问题

**Q: 双指点击没反应？**
A: 确保两根手指同时接触屏幕，时间间隔<100ms

**Q: 虚拟摇杆不灵敏？**
A: 尝试在更靠近左下角的位置触摸

**Q: 炮台无法放置？**
A: 检查是否有足够水晶，位置是否被占用

**Q: 如何快速退出建造模式？**
A: 再次双指点击，或点击ESC键（如果有键盘）
