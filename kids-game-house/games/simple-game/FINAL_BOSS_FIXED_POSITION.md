## BOSS战斗特性

### 📍 位置固定

```
┌──────────────────────┐
│                      │
│      👾 BOSS         │ ← 固定在y=80位置
│   (左右摆动)          │
│                      │
│                      │
│                      │
│   🚀 玩家            │
│                      │
└──────────────────────┘
```

- **X轴**：左右摆动（正弦波运动）
- **Y轴**：固定在80像素处，完全不移动
- **摆动幅度**：约±1.5像素/帧
- **摆动周期**：约6秒完成一次完整摆动

## 战斗时长估算

### 场景1：无道具（基础状态）

**参数：**
- BOSS血量：5000 HP
- 玩家伤害：2点/发
- 射击间隔：0.3秒

**计算：**
```
需要射击次数：5000 ÷ 2 = 2500次
战斗时长：2500 × 0.3 = 750秒 = 12.5分钟
```

**评价：** ⏰ 太长了！需要调整

---

### 场景2：有三连射道具

**参数：**
- BOSS血量：5000 HP
- 玩家伤害：2点/发 × 3发 = 6点/次
- 射击间隔：0.3秒

**计算：**
```
需要射击次数：5000 ÷ 6 ≈ 834次
战斗时长：834 × 0.3 ≈ 250秒 = 4.2分钟
```

**评价：** ⏰ 还是很长

---

### 场景3：有散射道具

```
战斗时长：238 × 0.3 ≈ 71秒 = 1.2分钟
```

**评价：** ✅ 合理
