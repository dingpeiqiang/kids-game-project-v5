# 💪 宝石消消乐 - 道具系统详解

## 🎯 道具概述

为了让游戏更加解压和爽快，我们添加了4种强大的道具，每种都有独特的功能和华丽的特效！

---

## 📦 道具列表

### 1. 🔄 洗牌 (Shuffle)
- **数量：** 3个
- **功能：** 重新排列棋盘上所有宝石
- **使用场景：** 
  - 陷入死胡同，没有可消除的组合
  - 想要重新布局寻找更好的机会
- **特效：** 30个彩色粒子四散飞溅
- **得分：** 无直接得分，但创造新机会

**使用技巧：**
- 当棋盘上没有明显匹配时使用
- 可以打破僵局，创造连锁反应

---

### 2. 🔨 锤子 (Hammer)
- **数量：** 2个
- **功能：** 消除单个指定的宝石
- **使用场景：**
  - 某个宝石阻碍了重要匹配
  - 需要快速清除特定位置
- **特效：** 15个定向粒子爆炸
- **得分：** 50分

**使用技巧：**
- 精准打击关键位置
- 配合其他宝石创造连击
- 高分值道具，谨慎使用

---

### 3. 💣 炸弹 (Bomb)
- **数量：** 1个
- **功能：** 消除3x3范围的宝石（中心+周围8个）
- **使用场景：**
  - 目标区域有多个潜在匹配
  - 需要大范围清理
  - 制造大规模连锁反应
- **特效：** 每个宝石8个粒子，共最多72个粒子
- **得分：** 30分 × 消除数量

**使用技巧：**
- 选择宝石密集的区域
- 瞄准可能触发连锁的位置
- 最多可消除9个宝石，最高270分！

---

### 4. 🌈 彩虹 (Rainbow)
- **数量：** 1个
- **功能：** 清除棋盘上所有同色宝石
- **使用场景：**
  - 某种颜色宝石较多时
  - 想要一次性大量消除
  - 追求高分和视觉冲击
- **特效：** 每个宝石6个彩虹色粒子，华丽多彩
- **得分：** 40分 × 消除数量

**使用技巧：**
- 观察棋盘，选择数量最多的颜色
- 理论上最多可消除10+个宝石
- 最高可达400+分，最强道具！
- 使用后不取消激活，可连续使用

---

## 🎮 使用方法

### 步骤1：激活血道具
点击底部的道具按钮，激活的道具会显示**金色边框**

### 步骤2：选择目标
- **洗牌：** 点击后立即生效
- **锤子/炸弹/彩虹：** 点击棋盘上的宝石作为目标

### 步骤3：享受特效
观看华丽的粒子爆炸和宝石消除动画！

### 步骤4：连锁反应
使用道具后会自动检测新的匹配，触发连锁消除

## 💡 策略建议

### 新手策略
1. **先观察：** 看看哪种道具最适合当前局面
2. **锤子优先：** 简单直接，容易掌握
3. **保留彩虹：** 等到有大量同色宝石时使用

### 进阶策略
1. **组合使用：** 先用炸弹清理，再用彩虹收割
2. **时机把握：** 在连击即将结束时使用道具延续
3. **最大化收益：** 彩虹道具瞄准数量最多的颜色

### 高手策略
1. **预判连锁：** 使用道具创造最大连锁反应
2. **分数优化：** 计算每种道具的预期得分
3. **节奏控制：** 合理分配道具，确保持续高分

## 🔧 技术实现

### 数据结构
```typescript
let powerups: {
  shuffle: number   // 3个
  hammer: number    // 2个
  bomb: number      // 1个
  rainbow: number   // 1个
}
```

### 核心函数
- `usePowerup()` - 使用道具主函数
- `shuffleBoard()` - 洗牌逻辑
- `eliminateSingle()` - 单个消除
- `eliminateArea()` - 范围消除
- `eliminateAllOfType()` - 同色清除

### 交互流程
1. 检测道具按钮点击
2. 激活血道具状态
3. 检测宝石点击
4. 执行道具效果
5. 触发粒子特效
6. 处理连锁反应
