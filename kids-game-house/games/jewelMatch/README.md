# 💎 宝石消消乐 (Jewel Match)

## 游戏说明

这是一个优化版的三消游戏，提供更解压、更爽快的游戏体验。

### 主要优化

1. **更大棋盘**：7列×9行（原6×8），+31.25%游戏区域
2. **更长时间**：60秒倒计时（原45秒）
3. **更高分数**：25分/宝石（原20分）
4. **新增宝石**：特殊钻石💎，共7种宝石类型
5. **华丽特效**：光晕效果、增强粒子、流畅动画
6. **强大道具**：4种解压道具，让游戏更爽快！

### 游戏玩法

- 点击选择一个宝石
- 再点击相邻宝石交换位置
- 3个或以上相同宝石连线消除
- 在60秒内获得最高分数
- **使用道具**：点击道具按钮激活，再点击宝石使用

### 道具系统 💪

| 道具 | 图标 | 数量 | 功能 | 得分 |
|------|------|------|------|------|
| 洗牌 | 🔄 | 3 | 重新排列所有宝石 | - |
| 锤子 | 🔨 | 2 | 消除单个宝石 | 50分 |
| 炸弹 | 💣 | 1 | 消除3x3范围 | 30分/个 |
| 彩虹 | 🌈 | 1 | 清除所有同色宝石 | 40分/个 |

**使用方式：**
1. 点击道具按钮激活血道具（金色边框）
2. 点击棋盘上的宝石使用道具
3. 享受华丽的消除特效！

### 技术实现

本游戏作为独立模块存在，可被集成到任何 Canvas 游戏中。

**导出接口：**
```typescript
export function initJewelMatch(
  engine: GameEngine,
  audioService: AudioService,
  onEnd: () => void
): void
```

**使用示例：**
```typescript
import { initJewelMatch } from './jewelMatch/src/game'

// 在游戏平台中调用
initJewelMatch(gameEngine, audioService, () => {
  console.log('游戏结束')
})
```
