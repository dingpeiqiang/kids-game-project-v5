## 💪 道具详解

### 🔄 洗牌 (Shuffle)
- **用途：** 打破僵局，重新布局
- **得分：** 无直接得分
- **特效：** 30个HSL彩色粒子四散
- **策略：** 没有可消除组合时使用

### 🔨 锤子 (Hammer)
- **用途：** 精准消除单个宝石
- **得分：** 50分
- **特效：** 15个圆周分布的爆炸粒子
- **策略：** 清除阻碍匹配的宝石

### 💣 炸弹 (Bomb)
- **用途：** 大范围消除（3x3）
- **得分：** 30分 × 消除数量（最高270分）
- **特效：** 每个宝石8个粒子，共最多72个
- **策略：** 瞄准宝石密集区域

### 🌈 彩虹 (Rainbow)
- **用途：** 清除所有同色宝石
- **得分：** 40分 × 消除数量（最高400+分）
- **特效：** 每个宝石6个彩虹色粒子
- **策略：** 选择数量最多的颜色
- **特殊：** 使用后不取消激活，可连续使用

## 📊 代码实现

### 新增变量
```typescript
let powerups: {
  shuffle: number   // 3
  hammer: number    // 2
  bomb: number      // 1
  rainbow: number   // 1
}

let activePowerup: string | null = null
```

### 核心函数
- `usePowerup(type, x?, y?)` - 使用道具主逻辑
- `shuffleBoard()` - 洗牌算法（Fisher-Yates）
- `eliminateSingle(x, y)` - 单个消除+特效
- `eliminateArea(cx, cy, radius)` - 范围消除
- `eliminateAllOfType(type)` - 同色清除
- `createShuffleEffect()` - 洗牌粒子特效

### 交互处理
- 检测道具按钮点击区域
- 切换激活状态
- 道具模式下的宝石点击处理
- 自动取消激活（彩虹除外）

### UI渲染
