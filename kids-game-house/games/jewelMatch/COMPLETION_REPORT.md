## 🚀 使用方式

### 集成到游戏平台

```typescript
import { initJewelMatch } from './games/jewelMatch/src/game'

// 在游戏平台中调用
initJewelMatch(
  gameEngine,      // 游戏引擎实例
  audioService,    // 音频服务实例
  () => {          // 游戏结束回调
    console.log('游戏结束')
  }
)
```

### 所需接口

**GameEngine:**
- `start()` - 开始游戏
- `endGame()` - 结束游戏
- `addScore(points, x?, y?)` - 添加分数
- `getScore()` - 获取当前分数
- `triggerRandomBuff()` - 触发随机增益

**AudioService:**
- `initOnGesture()` - 初始化音频
- `collect()` - 收集音效
- `win()` - 胜利音效
- `lose()` - 失败音效
