# 中文排版优化指南 / Chinese Typography Optimization Guide

## 🎯 已解决的排版问题

### 问题 1: 中文字符不显示
**原因**: 位图字体（bitmap font）只包含英文字符  
**解决**: 智能检测中文，自动切换到系统字体（微软雅黑/黑体）

### 问题 2: 中文换行错误
**原因**: `formatDialogue` 按空格分割单词，但中文没有空格  
**解决**: 检测中文后按字符数换行

### 问题 3: 行间距过小
**原因**: 默认行间距不适合中文阅读  
**解决**: 增加 `lineSpacing: 8px`

### 问题 4: 字体大小不匹配
**原因**: 位图字体的 size 和系统字体的 px 不对应  
**解决**: 建立字体大小映射表

## 📊 排版参数

### 字体大小映射

| 位图字体 Size | 系统字体 PX | 用途 |
|--------------|------------|------|
| 8            | 14px       | 小文本、标签 |
| 12           | 18px       | 中等文本 |
| 16           | 24px       | 标题、大文本 |

### 行间距设置

| 场景 | 行间距 | 说明 |
|------|--------|------|
| 单行文本 | 默认 | 不需要额外行间距 |
| 多行对话 | 8px | 改善可读性 |
| 长段落 | 10-12px | 更舒适的阅读体验 |

### 换行宽度

| 场景 | 最大宽度 | 说明 |
|------|---------|------|
| 对话框 | 400px | 标准对话宽度 |
| 角色描述 | 500px | 较长的背景故事 |
| 自定义 | options.maxWidth | 可手动指定 |

## 🔍 调试技巧

### 检查字体是否正确加载

在浏览器控制台运行：
```javascript
// 检查是否检测到中文
import { hasChinese } from './utilities/TextHelper';
console.log(hasChinese('你好'));  // true
console.log(hasChinese('Hello')); // false
```

### 调整字体大小

如果字体太大或太小，修改 `TextHelper.js` 中的映射：
```javascript
const fontSizeMap = {
  8: '12px',   // 调小
  12: '16px',
  16: '20px',
};
```

### 调整行间距

如果行间距不合适：
```javascript
const textStyle = {
  lineSpacing: 10,  // 增加到 10px
  // ...
};
```

### 调整换行宽度

如果文本太宽或太窄：
```javascript
createMultilineText(scene, x, y, text, size, {
  maxWidth: 600,  // 增加宽度
  // ...
});
```

## 📝 常见问题

### Q: 为什么中文字体看起来和英文不一样？
A: 中文字体使用系统字体（微软雅黑），英文使用像素风格的位图字体。这是为了保证中文能正确显示。如需统一风格，可以加载中文字体文件。

### Q: 如何调整特定文本的样式？
A: 使用 `options.style` 参数：
```javascript
createText(scene, x, y, text, size, {
  style: {
    fontWeight: 'bold',
    stroke: '#000000',
    strokeThickness: 2
  }
});
```

### Q: 文本显示模糊怎么办？
A: 确保字体大小是整数，避免缩放：
```javascript
// ✅ 好
fontSize: '16px'

// ❌ 不好
fontSize: '16.5px'
```
