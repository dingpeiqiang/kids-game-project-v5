# 贪吃蛇游戏资源检测更新

## 🎨 新增功能

为贪吃蛇游戏首页添加了**明显的资源检测loading提示**和**检测结果提示**。

## ✨ 功能特点

### 1. Loading弹窗
- **动画效果**：淡入 + 从底部滑入的动画
- **进度条**：绿色渐变进度条，带有闪光动画
- **检测步骤**（4个步骤）：
  - 1️⃣ 加载主题
  - 2️⃣ 检查图片
  - 3️⃣ 检查音频
  - 4️⃣ 准备就绪
- **视觉反馈**：
  - 未激活：灰色圆形
  - 激活中：绿色圆形，带脉冲动画
  - 已完成：蓝色圆形，显示对勾

### 2. 错误提示弹窗
- 检测失败时显示友好的错误信息
- 包含主题名称和缺少的资源列表
- 红色主题边框，更加醒目
- 点击"我知道了"关闭

## 📝 修改的文件

- `src/views/StartView.vue`

## 🚀 使用效果

当用户点击"开始游戏"按钮时：

1. 显示资源检测loading弹窗
2. 进度条从0%增长到100%
3. 4个检测步骤依次激活
4. 检测通过后自动关闭弹窗，跳转到难度选择页
5. 如果检测失败，显示错误弹窗，提示缺少哪些资源

## 📱 响应式设计

- 桌面端：4个步骤水平排列
- 移动端：步骤纵向排列，更加紧凑
