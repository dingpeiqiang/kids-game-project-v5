## 🐛 常见问题排查

### 问题 1：首页看不到主题选择按钮
- 检查 StartView.vue 是否导入 ThemeSelector
- 查看浏览器控制台是否有错误
- 确认 Vite 已热重载

### 问题 2：点击按钮没反应
- 检查控制台是否有 JavaScript 错误
- 确认 Pinia store 已正确初始化
- 验证 themeStore.init() 已调用

### 问题 3：主题切换后游戏画面未更新
- 检查 PhaserGame 是否监听 `theme-changed` 事件
- 验证 CSS 变量是否正确更新
- 查看游戏渲染循环是否正常

### 问题 4：刷新后主题丢失
- 检查 localStorage 是否保存成功
- 验证 loadTheme() 方法是否正常加载
- 查看 main.ts 中的 init() 调用时机
