# 贪吃蛇游戏主题选择功能实现

## 功能概述

为贪吃蛇游戏添加了主题选择功能，用户可以在**游戏首页**点击主题选择按钮（🎨图标），打开居中显示的主题选择列表，选择不同主题后游戏会立即加载对应主题的资源。

**使用场景：**
- ✅ **游戏首页**：在开始游戏前选择喜欢的主题（推荐）
- ~~游戏过程中~~：已移除右上角的主题按钮，专注于首页选择

## 实现内容

### 1. 新增文件

#### `src/stores/theme.ts`
- 使用 Pinia 管理主题状态
- 支持主题切换、持久化存储
- 动态应用主题到 CSS 变量
- 触发自定义事件通知游戏主题变更

**核心功能：**
- `currentThemeId`: 当前主题 ID
- `currentTheme`: 当前完整主题配置
- `isCustomTheme`: 是否使用自定义主题
- `themeList`: 主题列表
- `switchTheme()`: 切换主题
- `resetTheme()`: 重置为默认主题
- `init()`: 初始化主题

#### `src/components/ui/ThemeSelector.vue`
- 主题选择器组件
- 显示当前主题名称
- 点击打开主题选择面板
- 支持主题预览和快速切换

**UI 特性：**
- 🎨 调色板图标按钮
- 下拉式主题列表
- 主题预览色卡
- 选中状态标记（✓）
- 恢复默认按钮

### 3. 修改文件

#### `src/views/StartView.vue`
- 在游戏首页添加主题选择组件
- **音效开关和主题选择并排居中显示**
- 方便用户在开始游戏前选择主题

#### `src/components/ui/ThemeSelector.vue`
- **主题面板改为居中模态框设计**
- **添加半透明背景遮罩**
- **固定定位，屏幕居中，避免超出**
- **响应式尺寸：桌面端最大宽度 400px，移动端 95%**
- **最大高度 80vh，内容可滚动**

## 使用方法

### 在游戏首页选择主题（唯一入口）

1. **启动游戏**
   ```bash
   cd kids-game-house/snake-vue3
   npm run dev
   ```

2. **在首页选择主题**
   - 打开游戏首页
   - 在「开始游戏」按钮下方找到 🎨 主题选择按钮
   - **与音效开关并排居中显示**
   - 点击打开主题选择面板
   - **面板居中弹出，带半透明背景遮罩**
   - 选择你喜欢的主题
   - 点击「开始游戏」以选定的主题开始游戏

### 恢复默认主题
- 点击面板底部的「恢复默认」按钮
- 回到经典绿色主题

## 技术实现

### 主题应用机制

1. **CSS 变量更新**
   ```typescript
   root.style.setProperty('--theme-primary', theme.colors.primary)
   root.style.setProperty('--theme-secondary', theme.colors.secondary)
   // ... 更多变量
   ```

2. **自定义事件通知**
   ```typescript
   window.dispatchEvent(new CustomEvent('theme-changed', { 
     detail: theme 
   }))
   ```

3. **游戏资源重新渲染**
   - Phaser 游戏监听 `theme-changed` 事件
   - 事件触发时重新加载主题资源
   - 更新蛇、食物等游戏元素的外观

### 持久化存储

- 使用 localStorage 保存主题偏好
- 自动加载上次使用的主题
- 支持自定义主题保存
