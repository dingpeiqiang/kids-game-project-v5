## 🐛 常见问题排查

### 问题 1：障碍物不显示
**检查：**
```javascript
// 在 SnakeGame.vue 的 renderGame() 中添加调试
console.log('Obstacles:', gameStore.obstacles)
game.renderObstacles(gameStore.obstacles)
```

**可能原因：**
- 难度设置为 easy（应该为 0 个障碍物）
- GTRS 资源加载失败且 fallback 逻辑有问题
- obstacleGroup 未正确初始化

---

### 问题 2：食物类型单一
**检查：**
```javascript
// 在 game.ts 的 generateFood() 后添加调试
console.log('Generated food:', food.value)
```

**可能原因：**
- 随机数生成逻辑有误
- FOOD_TYPES 配置未正确更新
- 概率计算错误

---

### 问题 3：音频不播放
**检查：**
```javascript
// 在 PhaserGame.ts 的 playSound() 中添加调试
console.log('Playing sound:', soundKey, this.scene?.sound.get(actualKey))
```

**可能原因：**
- 浏览器自动播放策略限制（需要用户交互）
- GTRS 资源路径错误
- Phaser 音频解码失败
