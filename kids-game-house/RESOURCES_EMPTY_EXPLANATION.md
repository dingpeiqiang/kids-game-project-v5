## 💡 **何时使用 Resources 目录？**

### 📌 **适用场景**

1. **发现通用素材时**
   - 多个游戏使用相同的背景
   - 多个游戏使用相同的按钮/图标
   - → 提取到 `resources/` 避免重复

2. **开发新游戏时**
   - 需要通用素材库支持
   - → 从 `resources/` 获取已有素材

3. **统一风格时**
   - 需要所有游戏 UI 风格一致
   - → 在 `resources/` 中定义标准素材

### 🔄 **资源流动方向**

```
游戏专属资源          公共资源库
    ↓                    ↑
    └─→ 发现可复用 ──────┘
    
    新游戏 ←─ 从公共资源获取
```
