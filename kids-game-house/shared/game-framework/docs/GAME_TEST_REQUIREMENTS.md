## 性能测试

### 加载性能测试

#### 测试工具
- Chrome DevTools (Lighthouse)
- WebPageTest
- GTmetrix

---

#### 测试步骤

1. **清空缓存**
   ```
   Ctrl+Shift+Delete → 清空缓存
   ```

2. **打开 Performance 面板**
   ```
   F12 → Performance → Record
   ```

3. **刷新页面并录制**
   ```
   Ctrl+R → 等待加载完成 → Stop
   ```

4. **分析指标**
   - First Contentful Paint (FCP): < 1.5s
   - Largest Contentful Paint (LCP): < 2.5s
   - Time to Interactive (TTI): < 3.0s
   - Total Blocking Time (TBT): < 300ms

---

### 运行时性能测试

#### 帧率测试

**工具**: Chrome DevTools Performance 面板

**测试场景**:
1. **空闲场景**: 游戏运行但无操作
   - 预期：稳定 60 FPS

2. **复杂场景**: 大量物体同时运动
   - 预期：≥ 50 FPS

3. **极限场景**: 最大负载情况
   - 预期：≥ 30 FPS

---

#### 内存泄漏测试

**步骤**:
1. 打开 Memory 面板
2. Take Heap Snapshot（初始）
3. 进行游戏操作 5 分钟
4. Take Heap Snapshot（操作后）
5. Compare snapshots（比较）

**判断标准**:
- ✅ Detached DOM nodes < 100
- ✅ 内存增长 < 50MB
- ✅ 无持续增长趋势

---

### 压力测试

#### 长时间运行测试

**步骤**:
1. 启动游戏
2. 自动运行 30 分钟
3. 监控性能指标

**监控项**:
- FPS 稳定性
- 内存占用趋势
- CPU 占用趋势
- 错误日志

**合格标准**:
- ✅ 无崩溃
- ✅ FPS 波动 < 20%
- ✅ 内存增长 < 100MB

---

#### 多实例测试

**步骤**:
1. 同时打开 5 个游戏标签页
2. 同时运行游戏
3. 观察系统资源

**合格标准**:
- ✅ 单实例内存 < 256MB
- ✅ 总 CPU 占用 < 80%
- ✅ 无明显卡顿

## 安全测试

### XSS 攻击防护测试

#### 测试点

1. **URL 参数注入**
   ```
   http://localhost:3001?name=<script>alert('XSS')</script>
   ```
   **预期**: 脚本不执行，显示为文本

2. **输入框注入**
   ```
   在昵称输入框输入：<img src=x onerror=alert('XSS')>
   ```
   **预期**: 输入被转义或过滤

3. **LocalStorage 注入**
   ```javascript
   localStorage.setItem('token', '<script>alert(1)</script>')
   ```
   **预期**: 读取时正确处理

---

### CSRF 攻击防护测试

#### 测试示例

```typescript
// snake.test.ts
import { describe, it, expect } from 'vitest'
import { SnakeGame } from './SnakeGame'

describe('SnakeGame', () => {
  it('should initialize correctly', () => {
    const game = new SnakeGame()
    expect(game.score).toBe(0)
    expect(game.snake.length).toBe(3)
  })

  it('should move snake correctly', () => {
    const game = new SnakeGame()
    game.move('up')
    expect(game.snake[0].y).toBeLessThan(game.snake[1].y)
  })

  it('should increase score when eating food', () => {
    const game = new SnakeGame()
    game.eatFood()
    expect(game.score).toBeGreaterThan(0)
  })
})
```

```typescript
// snake.e2e.test.ts
import { test, expect } from '@playwright/test'

test('snake game basic flow', async ({ page }) => {
  // 1. 打开游戏页面
  await page.goto('http://localhost:3001')
  
  // 2. 等待游戏加载
  await expect(page.locator('.game-canvas')).toBeVisible()
  
  // 3. 开始游戏
  await page.click('#start-button')
  
  // 4. 控制蛇移动
  await page.keyboard.press('ArrowUp')
  await page.keyboard.press('ArrowRight')
  
  // 5. 验证分数变化
  const score = await page.locator('.score').textContent()
  expect(parseInt(score)).toBeGreaterThan(0)
  
  // 6. 验证游戏结束
  await expect(page.locator('.game-over')).toBeVisible({ timeout: 30000 })
})
```
