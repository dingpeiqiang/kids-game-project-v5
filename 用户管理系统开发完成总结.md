**后端实现**:
```java
@Override
@Transactional(rollbackFor = Exception.class)
public BaseUser register(UserRegisterDTO dto) {
    // 1. 创建用户
    BaseUser user = new BaseUser();
    // ...
    
    // 2. 初始化等级
    initializeUserLevel(user.getUserId());
    
    // 3. 如果是儿童，初始化疲劳点（10 点）
    if (user.getUserType() == 0) {
        initializeFatiguePoints(user.getUserId());
    }
    
    return user;
}
```

### **2. 疲劳点管理**

**后端实现**:
```java
// 定时任务：每天 06:00 重置疲劳点
@Scheduled(cron = "0 0 6 * * ?")
public void dailyReset() {
    List<BaseUser> kids = getKidUsers();
    for (BaseUser kid : kids) {
        kid.setFatiguePoints(10);
        baseUserMapper.updateById(kid);
        
        // 记录日志
        FatiguePointsLog log = new FatiguePointsLog();
        log.setChangeType(3);  // 3-每日重置
        log.setChangePoints(10);
        fatiguePointsLogMapper.insert(log);
    }
}
```

**前端调用**:
```typescript
// 获取疲劳点数
const points = await getFatiguePoints()

// 消耗疲劳点（游戏启动时）
await consumeFatiguePoints(gameId, points)

// 增加疲劳点（答题奖励）
await addFatiguePoints(questionId, points)
```

### **3. 监护关系管理**

**后端实现**:
```java
@PostMapping("/bind")
public Result<UserRelation> bindRelation(
    @RequestParam Long guardianUserId,
    @RequestParam Long kidUserId,
    @RequestParam String relationType,
    @RequestParam(required = false, defaultValue = "false") Boolean isPrimary,
    @RequestParam(required = false, defaultValue = "FULL") String permissionLevel
) {
    UserRelation relation = userRelationService.bindRelation(
        guardianUserId, kidUserId, relationType, isPrimary, permissionLevel
    );
    return Result.success(relation);
}
```

**前端调用**:
```typescript
import { bindRelation, unbindRelation, setPrimaryGuardian } from '@/api/relation'

// 解绑关系
```
