## 🔧 技术实现

### Sharp 库优势

相比纯 Node.js 手动绘制，Sharp 提供:

1. **SVG 渲染** - 使用矢量图形组合复杂图案
2. **高质量输出** - 抗锯齿、平滑边缘
3. **渐变支持** - 线性/径向渐变轻松实现
4. **文字渲染** - 可以直接添加文字和 emoji
5. **性能优秀** - C++ 底层，处理速度快

### SVG 绘图示例

```svg
<!-- 玩家坦克 -->
<svg width="48" height="48">
  <!-- 履带 -->
  <rect x="4" y="8" width="8" height="32" rx="2" fill="#15803d"/>
  
  <!-- 主体装甲 -->
  <rect x="14" y="12" width="20" height="24" rx="3" fill="#4ade80"/>
  
  <!-- 炮管 -->
  <rect x="20" y="2" width="8" height="16" rx="2" fill="#16a34a"/>
  
  <!-- 装饰 -->
  <circle cx="24" cy="22" r="4" fill="#166534"/>
</svg>
```

### 核心工具函数

```javascript
// 创建渐变矩形
createGradientRect(width, height, color1, color2)

// 创建圆形 (带渐变)
createCircle(size, color, withGradient)

// SVG 转 PNG
sharp(Buffer.from(svg)).png().toBuffer()
```

## 🎯 最佳实践

### 1. SVG 设计原则
- 使用相对坐标 (百分比)
- 合理使用 rx/ry 创建圆角
- 利用 defs 重用渐变定义
- 保持 viewBox 比例一致

### 2. 颜色搭配
- 同色系渐变 (主色→次色→强调色)
- 避免过多颜色 (每物体≤5 色)
- 注意对比度和可读性

### 3. 性能优化
- 复用相同的 SVG 模板
- 批量生成减少 I/O
- 适当控制图片尺寸
