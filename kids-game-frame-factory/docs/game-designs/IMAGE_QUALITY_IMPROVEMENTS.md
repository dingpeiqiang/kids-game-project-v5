## 🔧 技术亮点

### 1. 径向渐变算法
```javascript
const dist = Math.sqrt((x-cx)² + (y-cy)²);
const ratio = dist / maxRadius;

// 根据距离设置不同颜色
if (ratio < 0.3) return color1;
else if (ratio < 0.6) return color2;
else return color3;
```

### 2. 噪点增强真实感
```javascript
const noise = (Math.random() - 0.5) * 50;
return rgba(baseR + noise, baseG + noise, baseB + noise, alpha);
```

### 3. 多层结构构建
```javascript
// 核心层
if (dist < radius1) return mainColor;
// 中间层
else if (dist < radius2) return transitionColor;
// 外层的
else return edgeColor;
```

### 4. 程序化纹理
```javascript
// 砖块错缝
```

## 💡 设计原则

1. **层次感**: 每层物体都有 3+ 颜色层次
2. **质感优先**: 金属像金属，宝石像宝石
3. **动态效果**: 爆炸有粒子，道具有光泽
4. **色彩和谐**: 同色系渐变，避免突兀
5. **性能兼顾**: 算法简单但效果好
