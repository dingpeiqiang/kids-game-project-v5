# 《坦克大战》游戏设计文档

## 游戏概述
- **游戏名称**: 坦克大战 (Tank Battle)
- **游戏类型**: 动作射击 / 策略塔防
- **目标用户**: 8-14 岁儿童
- **核心玩法**: 控制坦克在地图上移动，消灭敌方坦克，保护基地，闯过关卡

## 技术规格
- **画布尺寸**: 720 × 1280 px
- **网格系统**: 24 × 36 格子 (每格 30×30 px)
- **帧率要求**: 60 FPS

## 游戏对象

### 玩家坦克
- **尺寸**: 48×48 px (占用 1.5 格)
- **移动方式**: 上下左右四个方向
- **移动速度**: 3 格/秒
- **旋转角度**: 0°, 90°, 180°, 270°
- **射击冷却**: 0.5 秒
- **生命值**: 3

### 敌方坦克
#### 类型 1: 普通坦克 (Basic Tank)
- **尺寸**: 48×48 px
- **颜色**: 红色
- **移动速度**: 2 格/秒
- **生命值**: 1
- **射击频率**: 1 次/秒
- **AI 行为**: 随机移动，发现玩家时追击

#### 类型 2: 快速坦克 (Fast Tank)
- **尺寸**: 42×42 px
- **颜色**: 黄色
- **移动速度**: 4 格/秒
- **生命值**: 1
- **射击频率**: 0.8 次/秒
- **AI 行为**: 快速接近玩家

#### 类型 3: 重型坦克 (Heavy Tank)
- **尺寸**: 54×54 px
- **颜色**: 深灰色
- **移动速度**: 1 格/秒
- **生命值**: 3
- **射击频率**: 0.5 次/秒
- **AI 行为**: 缓慢但火力强

### 子弹
- **玩家子弹**: 
  - 尺寸：12×12 px
  - 速度：8 格/秒
  - 伤害：1
  - 颜色：橙黄色
  
- **敌人子弹**: 
  - 尺寸：12×12 px
  - 速度：6 格/秒
  - 伤害：1
  - 颜色：红色

### 地形元素
#### 砖墙 (Brick Wall)
- **尺寸**: 30×30 px
- **特性**: 可被子弹摧毁
- **生命值**: 1
- **颜色**: 棕色

#### 钢墙 (Steel Wall)
- **尺寸**: 30×30 px
- **特性**: 不可摧毁
- **颜色**: 银灰色

#### 草地 (Grass)
- **尺寸**: 30×30 px
- **特性**: 坦克可以穿过，提供掩护
- **颜色**: 绿色

#### 水域 (Water)
- **尺寸**: 30×30 px
- **特性**: 坦克不能通过，子弹可以飞过
- **颜色**: 蓝色

#### 基地 (Base/Eagle)
- **尺寸**: 60×60 px (2×2 格)
- **特性**: 需要保护的目标，被摧毁则游戏结束
- **图标**: 鹰/星星标志

### 道具 (Power-up)
#### 类型 1: 三星 (Star) - 武器升级
- **尺寸**: 30×30 px
- **效果**: 提升子弹等级 (最高 3 级)
- **持续时间**: 永久 (当前生命)

#### 类型 2: 时钟 (Clock) - 时间冻结
- **尺寸**: 30×30 px
- **效果**: 冻结所有敌人 5 秒
- **持续时间**: 5 秒

#### 类型 3: 铲子 (Shovel) - 基地加固
- **尺寸**: 30×30 px
- **效果**: 基地周围钢墙保护 10 秒
- **持续时间**: 10 秒

#### 类型 4: 生命 (Life) - 额外生命
- **尺寸**: 30×30 px
- **效果**: 增加 1 条生命
- **持续时间**: 永久

## 场景元素

### 背景
- **主背景**: 深色军事风格渐变
- **网格线**: 淡色战术网格

### UI 界面
- **顶部信息栏**: 
  - 关卡数
  - 剩余敌人数量
  - 玩家生命数
  - 当前得分
  
- **暂停菜单**: 
  - 继续游戏
  - 重新开始
  - 返回主菜单

### 游戏状态
- **开始界面**: 游戏标题、开始按钮、关卡选择
- **游戏进行中**: 实时战斗
- **暂停界面**: 暂停菜单
- **游戏结束**: 失败画面、得分统计
- **胜利画面**: 通关显示、下一关按钮

## 关卡设计

### 第 1 关 (教学关)
- **敌人数量**: 5 辆普通坦克
- **地形**: 简单砖墙障碍
- **目标**: 熟悉操作

### 第 2-5 关
- **敌人数量**: 8-12 辆混合坦克
- **地形**: 增加钢墙、水域
- **Boss**: 无

### 第 6-10 关
- **敌人数量**: 12-15 辆包括重型坦克
- **地形**: 复杂地形，多路径
- **Boss**: 每 5 关出现 Boss 坦克

## 游戏模式

### 单人模式
- 闯关模式，共 20 关
- 存档功能
- 排行榜

### 本地双人合作
- 分屏或同屏合作
- 共同对抗 AI 敌人
- 友谊伤害可选

## 计分规则

| 行为 | 分数 |
|------|------|
| 击毁普通坦克 | 100 |
| 击毁快速坦克 | 200 |
| 击毁重型坦克 | 300 |
| 击毁 Boss 坦克 | 500 |
| 保护基地完好 | 500 (关卡结束) |
| 拾取道具 | 50 |
| 无伤通关 | 1000 (额外奖励) |
