## 🎯 核心特性

### 🎨 技术架构
- **前端框架**: Vue 3.4 + TypeScript
- **游戏引擎**: Phaser 3.70
- **状态管理**: Pinia 2.1
- **构建工具**: Vite 5.0
- **音频处理**: node-wav + FFmpeg
- **图片生成**: Canvas API

- **玩家坦克**: 4 个方向，流畅移动
- **敌方坦克**: 3 种类型 (普通/快速/重型)

- **道具系统**: 4 种 (升级/冻结/加固/生命)
- **关卡设计**: 规划 20 关
- **音效配乐**: 4 首 BGM + 7 首 SFX

## 🚀 快速启动

### 方式 1: 使用批处理 (推荐，Windows)

```bash
# 1. 生成资源
generate-resources.bat

# 2. 安装依赖并启动
npm install
npm run dev
```

### 方式 2: 手动执行

```bash
# 1. 进入游戏目录
cd kids-game-house/tank-battle-vue3

# 2. 安装主项目依赖
npm install

# 3. 安装脚本依赖
cd scripts
npm install
cd ..

# 4. 生成资源
node scripts/generate-resources.mjs

# 5. 启动开发服务器
npm run dev
```

浏览器访问：**http://localhost:3002**

### 🎨 自动化资源生成
- **Canvas 绘图**: 程序生成所有 PNG 图片，无需美术素材
- **音频合成**: 程序生成背景音乐和音效

- **严格校验**: 完全符合 GTRS v1.0.0 要求
- **双份输出**: GTRS.json 同时输出到 src 和 public
- **路径统一**: 所有资源使用 `/themes/default/` 前缀

- **Vue3 Composition API**: 最先进的 Vue 开发体验
