# 🚀 AudioTool 快速开始指南

## 📋 3 分钟快速上手

### 方式一：一键启动（推荐）

#### Windows 用户
```bash
# 1. 首次使用 - 安装依赖
双击 install.bat

# 2. 启动程序
双击 start.bat
```

### 方式二：命令行启动

```bash
# 1. 进入目录
cd kids-game-frame-factory/AudioTool

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行程序
python AudioTool.py
```

## 🎯 基础使用流程

### Step 1: 搜索音频 ⏱️ 30 秒
```
1. 打开程序后，在搜索框输入关键词
   推荐：消消乐 小游戏
   
2. 点击"🔍 开始搜索"

3. 等待搜索完成（进度条 100%）
```

### Step 2: 试听音频 ⏱️ 10 秒
```
方式一：双击列表中的任意音频
方式二：选中音频 → 点击"▶️ 试听"
```

### Step 3: 下载音频 ⏱️ 1 分钟
```
单个下载:
- 选中想要的音频
- 点击"💾 下载选中"

批量下载:
- 点击"📥 批量下载"
- 确认对话框
- 等待下载完成
```

### Step 4: 查看结果 ⏱️ 5 秒
```
点击"📂 打开目录"
→ 所有下载的音频保存在"小游戏音频"文件夹
```

## 🔧 故障排查

### ❌ 问题 1: 搜索不到音频
**解决方案**:
1. 更换推荐关键词（见上方表格）
2. 检查网络连接
3. 增加"最大爬取页面数"
4. 稍后再试（可能是网站临时维护）

### ❌ 问题 2: 播放失败
**解决方案**:
1. 检查 pygame 是否安装：`pip list | findstr pygame`
2. 重新安装 pygame: `pip install --upgrade pygame`
3. 尝试下载后使用其他播放器播放

### ❌ 问题 3: 下载失败
**解决方案**:
1. 检查网络连接
2. 尝试下载其他音频
3. 检查磁盘空间
4. 关闭杀毒软件/防火墙（临时）

### ❌ 问题 4: 程序闪退
**解决方案**:
1. 查看日志文件（控制台输出）
2. 检查 Python 版本（需要 3.8+）
3. 重新安装依赖：`pip install -r requirements.txt`

## 📊 完整工作流程示例

### 场景：为消消乐游戏收集音频素材

```
Step 1: 启动程序
└─ 双击 start.bat

Step 2: 搜索音频
├─ 输入关键词：消消乐 小游戏
├─ 设置最大结果数：30
└─ 点击"🔍 开始搜索"
   └─ 等待约 30-60 秒

Step 3: 筛选音频
├─ 浏览搜索结果列表
├─ 双击感兴趣的音频试听
└─ 标记喜欢的音频（记住编号）

Step 4: 批量下载
├─ 点击"📥 批量下载"
├─ 确认下载
└─ 等待 2-5 分钟

Step 5: 整理素材
├─ 点击"📂 打开目录"
├─ 将音频分类到不同文件夹:
│   ├─ BGM/        (背景音乐)
│   ├─ SFX/        (音效)
│   └─ UI/         (界面音效)
└─ 重命名文件（可选）

总计用时：约 5-10 分钟
获得素材：20-30 个音频文件
```

## 🎓 进阶技巧

### 技巧 1: 精准搜索
```
使用更具体的关键词:
- "消消乐 背景音乐" → 主要找 BGM
- "消消乐 消除音效" → 主要找音效
- "消消乐 ui 声音" → 找界面音效
```

### 技巧 2: 批量处理
```
1. 第一次搜索：消消乐 → 批量下载
2. 第二次搜索：贪吃蛇 → 批量下载
3. 第三次搜索：俄罗斯方块 → 批量下载
4. 统一整理所有下载的音频
```

### 技巧 3: 质量保证
```
1. 优先下载文件大小适中的音频
2. 试听时注意音质
3. 删除质量差的音频
4. 保留高质量的备用
```

### 技巧 4: 效率提升
```
使用多个搜索词组合:
- 消消乐 + 小游戏
- 休闲游戏 + 音效
- 手机游戏 + bgm

每次搜索后不清空列表，累积更多结果
```
