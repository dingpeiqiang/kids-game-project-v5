# 🎵 AudioTool - 小游戏音频爬取工具（优化版 v2.0）

一个功能强大的小游戏音频资源爬取和管理工具，支持搜索、试听、批量下载等功能。

## ✨ 特性亮点

### 🚀 核心功能
- **智能搜索**: 基于百度搜索，自动抓取小游戏音频资源
- **在线试听**: 直接播放搜索结果中的音频
- **批量下载**: 一键下载所有找到的音频
- **进度显示**: 实时展示搜索和下载进度
- **日志记录**: 完整的操作日志便于追踪

### 🛡️ 技术优势
- **重试机制**: 网络请求失败自动重试（最多 3 次）
- **超时控制**: 所有请求设置合理超时时间
- **异常处理**: 细粒度的错误捕获和友好提示
- **配置持久化**: 用户设置自动保存

### 💎 用户体验
- **现代化 UI**: 使用 clam 主题，美观大方
- **快捷操作**: 双击列表项直接播放
- **智能提示**: 丰富的状态提示和 emoji 图标
- **多任务**: 后台线程处理，界面不卡顿

## 🎯 使用指南

### ⚠️ 重要提示（2026-03-31）

由于现代网站普遍使用动态加载和严格的反爬虫机制，**自动爬取功能可能无法正常工作**。

**推荐用途**:
- ✅ 本地音频文件管理
- ✅ 音频试听和预览
- ✅ 批量下载已找到的音频链接
- ❌ 不建议作为主要的音频获取方式

**更好的替代方案**:
1. [OpenGameArt.org](https://opengameart.org/) - 免费游戏开发资源
2. [Kenney.nl](https://kenney.nl/assets) - 高质量游戏素材包
3. [Freesound.org](https://freesound.org/) - CC0 音效库

详细分析请阅读：[FINAL_SOLUTION.md](./FINAL_SOLUTION.md)

---

### 1. 搜索音频
1. 在搜索框输入关键词（推荐："消消乐 小游戏"）
2. 调整最大结果数（可选，默认 20）
3. 点击"🔍 开始搜索"
4. 等待搜索完成，查看结果列表

### 2. 试听音频
- **方式一**: 双击列表中的音频项
- **方式二**: 选中音频 → 点击"▶️ 试听"

### 3. 下载音频
#### 单个下载
1. 选中要下载的音频
2. 点击"💾 下载选中"
3. 等待下载完成

#### 批量下载
1. 点击"📥 批量下载"
2. 确认下载对话框
3. 等待批量下载完成

### 4. 其他功能
- **⏹ 停止**: 中断当前搜索任务
- **⏹ 停止播放**: 手动停止音频播放
- **📂 打开目录**: 快速访问保存文件夹
- **🗑 清空列表**: 清除当前搜索结果

## ⚙️ 配置说明

### 配置文件位置
```
小游戏音频/config.json
```

### 可配置参数
```json
{
  "save_root": "小游戏音频",           // 保存目录
  "audio_formats": [".mp3", ".wav", ".ogg", ".m4a"],  // 支持的格式
  "max_search_results": 20,            // 最大搜索结果数
  "max_pages_to_crawl": 5,             // 最大爬取页面数
  "request_timeout": 10,               // 请求超时时间 (秒)
  "max_retries": 3,                    // 最大重试次数
  "concurrent_threads": 3,             // 并发线程数
  "user_agent": "Mozilla/5.0..."       // User-Agent
}
```

### 性能调优建议

#### 快速搜索模式（适合测试）
```python
CONFIG["max_search_results"] = 10
CONFIG["max_pages_to_crawl"] = 3
CONFIG["request_timeout"] = 5
```

#### 深度搜索模式（适合批量采集）
```python
CONFIG["max_search_results"] = 50
CONFIG["max_pages_to_crawl"] = 10
CONFIG["max_retries"] = 5
```

## 🐛 常见问题

### Q: 搜索不到音频怎么办？
**A**: 尝试以下方法：
- 更换关键词（使用推荐关键词）
- 增加最大爬取页面数
- 检查网络连接
- 稍后再试（可能是网站临时问题）

### Q: 下载失败如何处理？
**A**: 可能原因：
- 链接失效 → 尝试其他音频
- 网络问题 → 检查连接
- 服务器限制 → 使用代理或稍后重试

### Q: 播放失败是什么原因？
**A**: 检查以下几点：
- pygame 是否正确安装
- 音频格式是否支持
- 临时文件权限问题

### Q: 如何批量管理下载的音频？
**A**: 使用内置功能：
- 点击"📂 打开目录"访问保存文件夹
- 使用文件管理器进行分类整理
- 定期清理不需要的音频
